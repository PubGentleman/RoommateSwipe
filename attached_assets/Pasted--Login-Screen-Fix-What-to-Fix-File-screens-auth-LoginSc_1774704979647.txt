## Fix 1 â€” Input Field Styling

The email and password fields are rendering with a light blue/white background. They need to match the dark app theme.

Find the `TextInput` components for email and password and update their styles:

```tsx
// Email input
<TextInput
  style={styles.input}
  placeholder="Enter your email"
  placeholderTextColor="rgba(255,255,255,0.35)"
  keyboardType="email-address"
  autoCapitalize="none"
  value={email}
  onChangeText={setEmail}
/>

// Password input
<TextInput
  style={styles.input}
  placeholder="Enter your password"
  placeholderTextColor="rgba(255,255,255,0.35)"
  secureTextEntry={!passwordVisible}
  value={password}
  onChangeText={setPassword}
/>
```

Update or replace the `input` style:

```ts
input: {
  flex: 1,
  color: '#FFFFFF',
  fontSize: 15,
  paddingVertical: 14,
  paddingHorizontal: 16,
  backgroundColor: 'transparent',
},
inputContainer: {
  flexDirection: 'row',
  alignItems: 'center',
  backgroundColor: 'rgba(255,255,255,0.07)',
  borderWidth: 1,
  borderColor: 'rgba(255,255,255,0.1)',
  borderRadius: 14,
  marginBottom: 14,
  paddingHorizontal: 4,
},
inputContainerFocused: {
  borderColor: '#F06464', // brand coral on focus
},
```

Wrap each `TextInput` in a `View` using `inputContainer` style, and track focus state to apply `inputContainerFocused`:

```tsx
const [emailFocused, setEmailFocused] = useState(false);
const [passwordFocused, setPasswordFocused] = useState(false);

<View style={[styles.inputContainer, emailFocused && styles.inputContainerFocused]}>
  <TextInput
    ...
    onFocus={() => setEmailFocused(true)}
    onBlur={() => setEmailFocused(false)}
  />
</View>

<View style={[styles.inputContainer, passwordFocused && styles.inputContainerFocused]}>
  <TextInput
    ...
    onFocus={() => setPasswordFocused(true)}
    onBlur={() => setPasswordFocused(false)}
  />
</View>
```

## Fix 3 â€” Add Password Visibility Toggle

Add a show/hide password button inside the password input container on the right side.

Add state:

```ts
const [passwordVisible, setPasswordVisible] = useState(false);
```

Add toggle button inside the password input container:

```tsx
<View style={[styles.inputContainer, passwordFocused && styles.inputContainerFocused]}>
  <Feather name="lock" size={16} color="rgba(255,255,255,0.35)" style={{ marginLeft: 12 }} />
  <TextInput
    style={styles.input}
    placeholder="Enter your password"
    placeholderTextColor="rgba(255,255,255,0.35)"
    secureTextEntry={!passwordVisible}
    value={password}
    onChangeText={setPassword}
    onFocus={() => setPasswordFocused(true)}
    onBlur={() => setPasswordFocused(false)}
  />
  <Pressable
    onPress={() => setPasswordVisible(!passwordVisible)}
    style={{ paddingHorizontal: 12 }}
    hitSlop={8}
  >
    <Feather
      name={passwordVisible ? 'eye-off' : 'eye'}
      size={18}
      color="rgba(255,255,255,0.4)"
    />
  </Pressable>
</View>
```
