**REPLIT PROMPT — Universal Date Picker Component**

Create a single reusable date picker component that is used everywhere in the app where a date needs to be entered. Replace all existing date inputs, text fields, or inline date entry with this component.

---

**CREATE THE COMPONENT**

Create `components/DatePickerModal.tsx` as a universal date picker used across the entire app.

The component accepts these props:

```ts
interface DatePickerModalProps {
  visible: boolean;
  onClose: () => void;
  onConfirm: (date: string) => void;
  mode: 'moveIn' | 'birthday' | 'availability' | 'general';
  title: string;
  initialDate?: string;          // ISO date string
  minDate?: string;              // ISO date string — no dates before this
  maxDate?: string;              // ISO date string — no dates after this
  showFlexible?: boolean;        // show "I'm Flexible" toggle (moveIn + availability only)
  tierLimit?: 30 | 90 | 180;    // locked date range for search (moveIn only)
  userPlan?: 'basic' | 'plus' | 'elite';
}
```

**VISUAL DESIGN**

Modal slides up from the bottom of the screen with a smooth spring animation.

Structure from top to bottom:
- Drag handle bar — centered, `rgba(255,255,255,0.2)`, 40px wide, 4px tall, rounded
- Title text — white, 18px bold, left aligned, e.g. "Select Move-In Date" or "Enter Your Birthday"
- Flexible toggle row — only shown when `showFlexible` is true
- Month navigation row — left arrow / **Month Year** / right arrow, coral arrows
- Day of week headers — Sun Mon Tue Wed Thu Fri Sat, small gray text
- Calendar grid — 6 rows of 7 days
- Upsell banner row — only shown in `moveIn` mode for Basic/Plus users
- Confirm and Clear buttons at the bottom

Style:
- Modal background: `#1a1a1a`
- Rounded top corners: `borderRadius: 24`
- Today's date: coral circle outline `#ff6b5b`
- Selected date: filled coral circle `#ff6b5b`, white text
- Past dates: `opacity: 0.3`, not tappable
- Locked tier dates: `opacity: 0.3` with a small 🔒 overlaid, not tappable
- Normal unselected dates: white text
- Confirm button: coral gradient pill, full width
- Clear button: ghost text link, centered below Confirm

**MODE BEHAVIOR**

**`birthday` mode:**
- `maxDate` automatically set to 18 years ago from today (must be 18+)
- `minDate` automatically set to 100 years ago
- No Flexible toggle
- No tier limit or upsell banner
- If user selects a date less than 18 years ago, show inline error: **"You must be at least 18 to use Roomdr"** in red below the calendar, do not allow confirm
- Start the calendar view on the birth year — show a **year selector** above the month navigation so users can jump to their birth year without tapping back 20+ years. Year selector is a horizontal scrollable strip of years

**`moveIn` mode:**
- `minDate` = today
- `maxDate` = today + tierLimit days (30, 90, or 180 based on plan)
- Flexible toggle visible
- Upsell banner visible for Basic and Plus users
- Locked dates beyond tier limit show 🔒 and tooltip on tap

**`availability` mode (host listing):**
- `minDate` = today
- No tier limit or upsell
- Flexible toggle visible
- No upsell banner

**`general` mode:**
- Uses whatever `minDate` and `maxDate` are passed in
- No Flexible toggle
- No upsell banner
- No tier limit

**DATE DISPLAY FORMAT**

Wherever a selected date is shown in the UI after selection, format it as:
- **"Mar 15, 2025"** — not ISO format, not slashes

Create a utility function `utils/dateUtils.ts`:

```ts
export const formatDate = (iso: string): string => {
  if (iso === 'flexible') return 'Flexible';
  const date = new Date(iso);
  return date.toLocaleDateString('en-US', { month: 'short', day: 'numeric', year: 'numeric' });
};

export const getMaxSearchDate = (plan: 'basic' | 'plus' | 'elite'): string => {
  const days = plan === 'elite' ? 180 : plan === 'plus' ? 90 : 30;
  const date = new Date();
  date.setDate(date.getDate() + days);
  return date.toISOString().split('T')[0];
};

export const isAtLeast18 = (dob: string): boolean => {
  const birth = new Date(dob);
  const today = new Date();
  const age = today.getFullYear() - birth.getFullYear();
  const m = today.getMonth() - birth.getMonth();
  return age > 18 || (age === 18 && m >= 0 && today.getDate() >= birth.getDate());
};
```
