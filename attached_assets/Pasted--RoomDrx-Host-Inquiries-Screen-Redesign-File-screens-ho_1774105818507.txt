# RoomDrx — Host Inquiries Screen Redesign

## File: `screens/host/HostInquiriesScreen.tsx`

---

## Issues Being Fixed

1. **Massive dead space** — filter tabs sit at the top then nothing; bottom 80% of the screen is empty even before scrolling
2. **No summary stats** — host has no idea at a glance how many pending/accepted/passed inquiries exist; must tap each filter to find out
3. **Filter tabs are plain text** — no count badges, no visual state difference beyond the coral pill
4. **AI button floats awkwardly** — coral robot icon sits in the top-right corner with no context label, looks like a stray FAB
5. **Empty state is cold** — generic heart icon with grey text; no action offered; user is stuck
6. **No inquiry card design** — when inquiries exist, cards need to feel premium (photo, match score, action buttons)

---

## Change 2 — Filter Tabs: Segmented Control with Count Badges

Replace the plain text tabs with a proper segmented pill control that shows live counts.

```tsx
// Replace the existing filter tab row with:
<View style={styles.filterWrap}>
  <ScrollView
    horizontal
    showsHorizontalScrollIndicator={false}
    contentContainerStyle={styles.filterScroll}
  >
    {(['all', 'pending', 'accepted', 'passed'] as const).map((tab) => {
      const label = tab.charAt(0).toUpperCase() + tab.slice(1);
      const count = tab === 'all' ? allCount : tab === 'pending' ? pendingCount : tab === 'accepted' ? acceptedCount : passedCount;
      const isActive = activeFilter === tab;
      return (
        <Pressable
          key={tab}
          style={[styles.filterTab, isActive && styles.filterTabActive]}
          onPress={() => setActiveFilter(tab)}
        >
          <Text style={[styles.filterTabText, isActive && styles.filterTabTextActive]}>
            {label}
          </Text>
          {count > 0 && (
            <View style={[styles.filterBadge, isActive && styles.filterBadgeActive]}>
              <Text style={[styles.filterBadgeText, isActive && styles.filterBadgeTextActive]}>
                {count}
              </Text>
            </View>
          )}
        </Pressable>
      );
    })}
  </ScrollView>
</View>
```

## Change 5 — Inquiry Card Design (for when data exists)

When inquiries are present, each card should show the renter's photo, name, match score, listing name, and two action buttons.

```tsx
// Inquiry card component (inside FlatList renderItem):
const renderInquiry = ({ item }: { item: Inquiry }) => (
  <View style={styles.inquiryCard}>
    {/* Left: Avatar + Match Badge */}
    <View style={styles.inquiryAvatarWrap}>
      {item.renter.photo ? (
        <Image source={{ uri: item.renter.photo }} style={styles.inquiryAvatar} />
      ) : (
        <LinearGradient colors={['#667eea', '#764ba2']} style={styles.inquiryAvatar}>
          <Text style={styles.inquiryAvatarLetter}>{item.renter.name[0]}</Text>
        </LinearGradient>
      )}
      <View style={[
        styles.matchBadge,
        { backgroundColor: item.matchScore >= 70 ? '#2ecc71' : item.matchScore >= 40 ? '#f39c12' : '#e74c3c' }
      ]}>
        <Text style={styles.matchBadgeText}>{item.matchScore}%</Text>
      </View>
    </View>

    {/* Center: Info */}
    <View style={styles.inquiryInfo}>
      <Text style={styles.inquiryName} numberOfLines={1}>{item.renter.name}</Text>
      <Text style={styles.inquiryListing} numberOfLines={1}>
        <Feather name="home" size={11} color="rgba(255,255,255,0.3)" />
        {' '}{item.listingTitle}
      </Text>
      <Text style={styles.inquiryTime}>{formatTimeAgo(item.createdAt)}</Text>
    </View>

    {/* Right: Actions */}
    <View style={styles.inquiryActions}>
      {item.status === 'pending' ? (
        <>
          <Pressable style={styles.actionPass} onPress={() => handlePass(item.id)}>
            <Feather name="x" size={16} color="rgba(255,255,255,0.4)" />
          </Pressable>
          <Pressable style={styles.actionAccept} onPress={() => handleAccept(item.id)}>
            <Feather name="check" size={16} color="#fff" />
          </Pressable>
        </>
      ) : (
        <Pressable style={styles.actionMessage} onPress={() => handleMessage(item)}>
          <Feather name="message-circle" size={16} color="#fff" />
        </Pressable>
      )}
    </View>
  </View>
);
```
