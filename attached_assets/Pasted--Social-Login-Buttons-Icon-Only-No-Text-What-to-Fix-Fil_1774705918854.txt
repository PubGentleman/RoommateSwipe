# Social Login Buttons — Icon Only (No Text)

## What to Fix

**File:** `screens/auth/LoginScreen.tsx` (or wherever the Google and Apple sign-in buttons are rendered — search for "Google" to find the exact file)

---

## Step 1 — Download the Google Logo Asset

Download the official Google "G" logo PNG and save it to the project assets:

```bash
curl -o assets/icons/google.png "https://upload.wikimedia.org/wikipedia/commons/thumb/c/c1/Google_%22G%22_logo.svg/768px-Google_%22G%22_logo.svg.png"
```

Or use the `expo-asset` approach and reference it directly from a URL if you prefer not to save it locally.

---

## Step 2 — Replace Both Buttons with Icon-Only Versions

Find the Google and Apple button components and replace them with icon-only square buttons — no text labels:

```tsx
import { Image, Platform } from 'react-native';
import { Ionicons } from '@expo/vector-icons';

{/* Social buttons row */}
<View style={styles.socialRow}>

  {/* Google — icon only */}
  <Pressable style={styles.socialIconButton} onPress={handleGoogleSignIn}>
    <Image
      source={require('../../assets/icons/google.png')}
      style={styles.googleIcon}
      resizeMode="contain"
    />
  </Pressable>

  {/* Apple — icon only, iOS only */}
  {Platform.OS === 'ios' && (
    <Pressable style={styles.socialIconButton} onPress={handleAppleSignIn}>
      <Ionicons name="logo-apple" size={26} color="#FFFFFF" />
    </Pressable>
  )}

</View>
```
