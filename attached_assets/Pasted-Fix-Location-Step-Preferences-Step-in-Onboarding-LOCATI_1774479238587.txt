LOCATION STEP — Cascading 3-tier selector (4-tier for NYC)
Replace the city chip grid entirely. Build a cascading location selector where each tier unlocks only after the one above it is filled. Each tier is a rounded card with an icon on the left, a small label above the value, and a short vertical gradient connector line between cards. Locked tiers are dimmed at opacity 0.45. Tapping a locked tier triggers a gentle horizontal shake animation.
Standard flow (all cities except NYC):
Tier 1 — State
Dropdown picker with all 50 US states. On selection, show a green checkmark on the card and unlock Tier 2.
Tier 2 — City
Text input with autocomplete. As the user types, show a dropdown filtered to cities within the selected state only. On selection, show a green checkmark and unlock Tier 3.
Tier 3 — Neighborhood (optional)
Text input with autocomplete filtered to neighborhoods within the selected city. Use a hardcoded dataset of neighborhoods for the top 20 US cities. For cities not in the dataset, allow free text entry. Has a "Skip neighborhood" link below the Continue button.
