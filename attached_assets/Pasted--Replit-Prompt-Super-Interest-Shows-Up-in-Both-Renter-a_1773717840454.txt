## OVERVIEW

1. The **renter's** Messages tab — so they can track it and know it was delivered

Super Interest is a paid feature so it must look and feel different from a regular inquiry on both sides.

### 3. `screens/shared/MessagesScreen.tsx` — Pin Super Interest threads to the top

Super Interest threads should always appear at the top of the "LISTING INQUIRIES" section, above regular pending inquiries.

**Find where inquiry threads are sorted/rendered** and sort them so Super Interest comes first:

```tsx
const inquiryThreads = conversations
  .filter(c => c.isInquiryThread)
  .sort((a, b) => {
    // Super Interest always floats to top
    if (a.isSuperInterest && !b.isSuperInterest) return -1;
    if (!a.isSuperInterest && b.isSuperInterest) return 1;
    // Then sort by most recent
    return new Date(b.lastMessageTime).getTime() - new Date(a.lastMessageTime).getTime();
  });
```
