```ts
// Sort order: Featured boosted → Plain boosted → Normal listings
listings.sort((a, b) => {
  const aFeatured = isListingBoosted(a) && a.listingBoost?.includesFeaturedBadge;
  const aBoosted = isListingBoosted(a) && !a.listingBoost?.includesFeaturedBadge;
  const bFeatured = isListingBoosted(b) && b.listingBoost?.includesFeaturedBadge;
  const bBoosted = isListingBoosted(b) && !b.listingBoost?.includesFeaturedBadge;

  if (aFeatured && !bFeatured) return -1;
  if (!aFeatured && bFeatured) return 1;
  if (aBoosted && !bBoosted) return -1;
  if (!aBoosted && bBoosted) return 1;
  return 0;
});
```

Featured listings always rank above plain-boosted listings, which always rank above normal listings.
