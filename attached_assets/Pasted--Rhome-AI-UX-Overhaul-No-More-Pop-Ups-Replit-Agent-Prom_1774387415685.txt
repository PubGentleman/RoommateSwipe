# Rhome — AI UX Overhaul: No More Pop-Ups
## Replit Agent Prompt

---

## Overview

The current AI interactions interrupt the user with modal pop-ups. This prompt replaces every AI pop-up with inline, non-intrusive surfaces that feel native to the swipe experience. Users are never pulled out of their flow — Claude's suggestions appear in context and can be ignored or engaged with on the user's own terms.

Three changes:

1. **Refinement questions** → become native cards injected into the swipe deck between profiles
2. **AI insight floating button** → replaced with a slim contextual banner above the deck
3. **Any remaining AI modal triggers** → removed; users reach the AI assistant via its own tab

---

## Part 1 — Refinement Question as a Deck Card

### How it works

Instead of a modal overlay interrupting the user mid-swipe, the refinement question appears as a special card in the swipe deck — between real profile cards. It looks polished and intentional, not like an interruption. The user can swipe up to answer, swipe left/right to skip (dismisses for 24 hours), or tap an answer option directly.

### Step 1 — Create `AIQuestionCard` component

Create file `components/AIQuestionCard.tsx`:

```tsx
import React, { useState } from 'react';
import {
  View, Text, TouchableOpacity, Animated, StyleSheet
} from 'react-native';
import { Ionicons } from '@expo/vector-icons';
import { LinearGradient } from 'expo-linear-gradient';
import { useTheme } from '../context/ThemeContext';
import type { RefinementQuestion } from '../utils/refinementQuestions';

interface Props {
  question: RefinementQuestion;
  onAnswer: (questionId: string, value: string) => void;
  onSkip: () => void;
}

export function AIQuestionCard({ question, onAnswer, onSkip }: Props) {
  const { theme } = useTheme();
  const [selected, setSelected] = useState<string | null>(null);
  const [answered, setAnswered] = useState(false);

  async function handleAnswer(value: string) {
    if (answered) return;
    setSelected(value);
    setAnswered(true);
    // Small delay so user sees the selection before card advances
    setTimeout(() => {
      onAnswer(question.id, value);
    }, 600);
  }

  return (
    <View style={[styles.card, { backgroundColor: theme.background, borderColor: theme.border }]}>
      {/* Top gradient accent */}
      <LinearGradient
        colors={[theme.primary + 'CC', theme.primary + '44']}
        start={{ x: 0, y: 0 }}
        end={{ x: 1, y: 1 }}
        style={styles.gradientHeader}
      >
        <View style={styles.headerRow}>
          <View style={styles.aiLabel}>
            <Ionicons name="sparkles" size={14} color="#fff" />
            <Text style={styles.aiLabelText}>Rhome AI</Text>
          </View>
          <TouchableOpacity onPress={onSkip} hitSlop={{ top: 10, bottom: 10, left: 10, right: 10 }}>
            <Text style={styles.skipText}>Skip for now</Text>
          </TouchableOpacity>
        </View>

        <Text style={styles.questionText}>{question.aiMessage}</Text>
      </LinearGradient>

      {/* Answer options */}
      <View style={styles.optionsContainer}>
        {question.options.map((opt) => {
          const isSelected = selected === opt.value;
          const isOther = answered && !isSelected;

          return (
            <TouchableOpacity
              key={opt.value}
              onPress={() => handleAnswer(opt.value)}
              disabled={answered}
              style={[
                styles.option,
                {
                  backgroundColor: isSelected ? theme.primary + '20' : theme.backgroundSecondary,
                  borderColor: isSelected ? theme.primary : theme.border,
                  opacity: isOther ? 0.4 : 1,
                },
              ]}
              activeOpacity={0.75}
            >
              <Text style={styles.optionEmoji}>{opt.emoji}</Text>
              <Text style={[
                styles.optionLabel,
                { color: isSelected ? theme.primary : theme.text, fontWeight: isSelected ? '700' : '500' }
              ]}>
                {opt.label}
              </Text>
              {isSelected && (
                <Ionicons name="checkmark-circle" size={18} color={theme.primary} style={{ marginLeft: 'auto' }} />
              )}
            </TouchableOpacity>
          );
        })}
      </View>

      {/* Post-answer message */}
      {answered && (
        <View style={styles.followUp}>
          <Ionicons name="checkmark-circle" size={16} color={theme.primary} />
          <Text style={[styles.followUpText, { color: theme.textSecondary }]}>
            {question.followUpMessage}
          </Text>
        </View>
      )}

      {/* Bottom hint */}
      {!answered && (
        <Text style={[styles.hint, { color: theme.textSecondary }]}>
          Swipe left to skip • Your answer improves your matches
        </Text>
      )}
    </View>
  );
}

const styles = StyleSheet.create({
  card: {
    width: '100%',
    borderRadius: 20,
    overflow: 'hidden',
    borderWidth: 1,
    shadowColor: '#000',
    shadowOffset: { width: 0, height: 4 },
    shadowOpacity: 0.08,
    shadowRadius: 12,
    elevation: 4,
  },
  gradientHeader: {
    padding: 20,

  },
  skipText: {
    color: 'rgba(255,255,255,0.75)',

  },
  questionText: {
    color: '#fff',
```

### Step 2 — Inject the card into the swipe deck

In `screens/renter/RoommatesScreen.tsx`:

**Find** the `profiles` state array that feeds the swipe deck. It is used something like:

```tsx
const [profiles, setProfiles] = useState<Profile[]>([]);
```

**Add** state for the pending refinement question:

```tsx
const [pendingQuestion, setPendingQuestion] = useState<RefinementQuestion | null>(null);
const [questionInjected, setQuestionInjected] = useState(false);
```

**Add** this function that checks whether to inject a question after each swipe:

```tsx
async function checkRefinementTrigger() {
  const { rightSwipes, totalSwipes, personalityAnswers, allAnswered } = await getSwipeStats(); // use existing stats logic
  const should = await shouldShowRefinementQuestion(rightSwipes, totalSwipes, personalityAnswers, allAnswered);

  if (should && !questionInjected) {
    const asked = await getQuestionsAsked();
    const next = getNextRefinementQuestion(asked, personalityAnswers);
    if (next) {
      setPendingQuestion(next);
      setQuestionInjected(true);
    }
  }
}
```

Call `checkRefinementTrigger()` inside the `handleSwipe` function after each swipe action (right after the swipe is recorded, before moving to the next card).

**Update the deck render logic.** Find where the swipe deck renders its top card (typically `profiles[currentIndex]`) and add the question card injection:

```tsx
// Determine what to show as the current "card"
const currentCard = (() => {
  // If we have a pending question and we've hit the injection point, show it
  if (pendingQuestion && questionInjected && currentIndex > 0 && currentIndex % 5 === 0) {
    return { type: 'question', data: pendingQuestion };
  }
  if (profiles[currentIndex]) {
    return { type: 'profile', data: profiles[currentIndex] };
  }
  return null;
})();
```

Then in the render:

```tsx
{currentCard?.type === 'question' ? (
  <AIQuestionCard
    question={currentCard.data as RefinementQuestion}
    onAnswer={async (questionId, value) => {
      await markQuestionAsked(questionId);
      await saveRefinementAnswer(questionId, value); // updates profile + AI memory
      setPendingQuestion(null);
      setQuestionInjected(false);
    }}
    onSkip={() => {
      setPendingQuestion(null);
      setQuestionInjected(false);
      resetRefinementCooldown();
    }}
  />
) : currentCard?.type === 'profile' ? (
  // existing profile card render
  <SwipeCard profile={currentCard.data as Profile} ... />
) : (
  // existing empty state
  <EmptyDeckView />
)}
```

**Add** `saveRefinementAnswer` to `services/discoverService.ts`:

```typescript
export async function saveRefinementAnswer(questionId: string, value: string): Promise<void> {
  const { data: { user } } = await supabase.auth.getUser();
  if (!user) return;

  // Update personality_answers on profile
  const { data: profile } = await supabase
    .from('profiles')
    .select('personality_answers')
    .eq('id', user.id)
    .single();

  const updated = {
    ...(profile?.personality_answers || {}),
    [questionId]: value,
  };

  await supabase
    .from('profiles')
    .update({ personality_answers: updated })
    .eq('id', user.id);

  // Save to AI memory
  await supabase
    .from('user_ai_memory')
    .insert({
      user_id: user.id,
      memory_text: `Refinement answer: ${questionId} = ${value}`,
      memory_type: 'refinement',
    });
}
```

## Part 2 — Replace Floating AI Button with Contextual Banner

### Remove

In `RoommatesScreen.tsx`, find the floating AI button — it will look something like:

```tsx
<TouchableOpacity style={styles.floatingAIButton} onPress={() => navigation.navigate('AIAssistant')}>
  <Ionicons name="sparkles" ... />
</TouchableOpacity>
```

Remove this component and its associated styles.

### Add — `AIInsightBanner`

Create file `components/AIInsightBanner.tsx`:

```tsx
import React, { useEffect, useState } from 'react';
import { View, Text, TouchableOpacity, Animated, StyleSheet } from 'react-native';
import { Ionicons } from '@expo/vector-icons';
import { useTheme } from '../context/ThemeContext';
import { getSwipeInsight } from '../services/aiService'; // existing function from AI improvements prompt

interface Props {
  onPress: () => void; // navigates to AI assistant tab
}

export function AIInsightBanner({ onPress }: Props) {
  const { theme } = useTheme();
  const [insight, setInsight] = useState<string | null>(null);
  const [visible, setVisible] = useState(false);
  const slideAnim = new Animated.Value(-60);

  useEffect(() => {
    loadInsight();
  }, []);

  async function loadInsight() {
    const text = await getSwipeInsight();
    if (text) {
      setInsight(text);
      setVisible(true);
      Animated.spring(slideAnim, {
        toValue: 0,
        useNativeDriver: true,
        tension: 80,
        friction: 10,
      }).start();
    }
  }

  function dismiss() {
    Animated.timing(slideAnim, {
      toValue: -60,
      duration: 250,
      useNativeDriver: true,
    }).start(() => setVisible(false));
  }

  if (!visible || !insight) return null;

  return (
    <Animated.View style={[
      styles.banner,
      {
        backgroundColor: theme.backgroundSecondary,
        borderColor: theme.border,
        transform: [{ translateY: slideAnim }],
      }
    ]}>
      <Ionicons name="sparkles" size={14} color={theme.primary} />
      <Text
        style={[styles.text, { color: theme.text }]}
        numberOfLines={1}
        ellipsizeMode="tail"
      >
        {insight}
      </Text>
      <TouchableOpacity onPress={onPress} style={[styles.actionBtn, { backgroundColor: theme.primary }]}>
        <Text style={styles.actionText}>Ask AI</Text>
      </TouchableOpacity>
      <TouchableOpacity onPress={dismiss} hitSlop={{ top: 10, bottom: 10, left: 10, right: 10 }}>
        <Ionicons name="close" size={16} color={theme.textSecondary} />
      </TouchableOpacity>
    </Animated.View>
  );
}

const styles = StyleSheet.create({
  banner: {
    flexDirection: 'row',
    alignItems: 'center',
    gap: 8,
    paddingHorizontal: 14,
    paddingVertical: 10,
    marginHorizontal: 16,
    marginBottom: 10,
    borderRadius: 14,
    borderWidth: 1,
  },
  text: {
    flex: 1,
    fontSize: 13,
    fontWeight: '500',
  },
  actionBtn: {
    paddingHorizontal: 12,
    paddingVertical: 5,
    borderRadius: 100,
  },
  actionText: {
    color: '#fff',
    fontSize: 12,
    fontWeight: '700',
  },
});
```

**Place** `AIInsightBanner` in `RoommatesScreen.tsx` directly above the swipe deck, below the screen header:

```tsx
<AIInsightBanner onPress={() => navigation.navigate('AIAssistant')} />
```

The banner slides in from the top when Claude has a real insight (based on swipe behavior), shows a one-liner with an "Ask AI" button, and has an X to dismiss. It never appears if `getSwipeInsight()` returns null — so it only shows when there's something genuinely useful to say.

## Part 3 — Clean Up Any Other AI Modals

Search the entire codebase for these patterns and check each one:

```
grep -r "Modal" --include="*.tsx" | grep -i "ai\|refinement\|question\|claude\|insight"
```

For any AI-related modal found that isn't the "Why This Match?" bottom sheet (which stays — it's user-triggered), remove the modal and either replace with a banner or remove entirely.

## Summary of Changes

| Before | After |
|---|---|
| Refinement question pops up as modal mid-swipe | Appears as a native card in the deck between profiles |
| Floating ✨ button launches full AI chat screen | Slim banner slides in above deck with one-liner insight |
| All AI interactions interrupt user flow | All AI surfaces are passive — user engages when ready |
| Skip = dismiss the modal | Skip = swipe left on the question card (feels natural) |
| Answer = tap inside a modal | Answer = tap inside the card (same gesture as normal use) |

The AI assistant chat tab remains for users who want deep conversations — it's just no longer pushed on users via a floating button.
