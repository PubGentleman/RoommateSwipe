# Replit Prompt — Allow Limited Messaging Without a Match (Open Cold Messaging)

## Overview
Currently non-Elite users see a "Match Required" paywall when trying to message someone they haven't mutually matched with. This change removes that hard wall and gives all users a daily cold message allowance instead. Matched conversations remain unlimited for all plans.

## New Cold Message Rules (Renter Plans)

| Plan | Cold messages per day (non-matched) | Matched messages |
|------|-------------------------------------|-----------------|
| Basic | 3 per day | Unlimited |
| Plus | 10 per day | Unlimited |
| Elite | Unlimited | Unlimited |

Cold message count resets at midnight. If the user hits their daily limit, show a friendly Alert — not a paywall modal.

### 1. `utils/messagingUtils.ts` — Add cold message daily limit helpers

Add the following constants and functions:

```ts
import AsyncStorage from '@react-native-async-storage/async-storage';

export const COLD_MESSAGE_LIMITS: Record<string, number> = {
  basic: 3,
  plus: 10,
  elite: Infinity,
};

export function getDailyColdMessageLimit(plan: string): number {
  return COLD_MESSAGE_LIMITS[plan] ?? 3;
}

export async function getDailyColdMessageCount(userId: string): Promise<number> {
  const today = new Date().toISOString().split('T')[0]; // YYYY-MM-DD
  const key = `cold_msg_count_${userId}_${today}`;
  const stored = await AsyncStorage.getItem(key);
  return stored ? parseInt(stored, 10) : 0;
}

export async function incrementDailyColdMessageCount(userId: string): Promise<void> {
  const today = new Date().toISOString().split('T')[0];
  const key = `cold_msg_count_${userId}_${today}`;
  const current = await getDailyColdMessageCount(userId);
  await AsyncStorage.setItem(key, String(current + 1));
}
```

### 2. `contexts/AuthContext.tsx` — Update `canSendColdMessage()`

Find the existing `canSendColdMessage` function and replace it so all plans can cold message, just with different daily limits:

```ts
const canSendColdMessage = async (): Promise<{ canSend: boolean; reason?: string; remaining: number }> => {
  if (!user) return { canSend: false, reason: 'Not logged in', remaining: 0 };

  const plan = user.subscription?.plan || 'basic';
  const limit = getDailyColdMessageLimit(plan);

  if (limit === Infinity) return { canSend: true, remaining: Infinity };

  const count = await getDailyColdMessageCount(user.id);
  const remaining = Math.max(0, limit - count);

  if (remaining <= 0) {
    return {
      canSend: false,
      reason: `You've used all ${limit} free messages for today. Resets at midnight.`,
      remaining: 0,
    };
  }

  return { canSend: true, remaining };
};
```

Make sure `getDailyColdMessageLimit` and `getDailyColdMessageCount` are imported from `../utils/messagingUtils`.

### 3. `screens/renter/RoommatesScreen.tsx` — Replace "Match Required" modal with daily limit flow

#### 3a. Update `handleMessageClick`

Replace the current logic that blocks non-Elite users with a "Match Required" modal:

```ts
const handleMessageClick = async () => {
  if (!currentProfile || !user) return;

  const matches = await StorageService.getMatches();
  const hasMatch = matches.some(m =>
    (m.userId1 === user.id && m.userId2 === currentProfile.id) ||
    (m.userId2 === user.id && m.userId1 === currentProfile.id)
  );

  if (hasMatch) {
    // Matched — open chat directly, no cold message limit applies
    handleSendDirectMessage(false);
  } else {
    // Not matched — check daily cold message limit
    const coldCheck = await canSendColdMessage();
    if (!coldCheck.canSend) {
      Alert.alert(
        'Daily Limit Reached',
        coldCheck.reason || `You've used all your messages for today. Resets at midnight.`,
        [
          {
            text: 'Upgrade for More',
            onPress: () => (navigation as any).getParent()?.navigate('Profile', { screen: 'Plans' }),
          },
          { text: 'OK', style: 'cancel' },
        ]
      );
      return;
    }
    // Still has cold messages remaining — open chat
    handleSendDirectMessage(true);
  }
};
```

## What Does NOT Change

- Matched conversations are unlimited for all plans — no cold message limit applies
- Elite users keep unlimited cold messages as a premium perk
- The regular daily message limit (for matched chats) is unchanged
- Super Interest, interest cards, and the Explore/listing inquiry flow are unchanged
- Host subscription plans (Free, Starter, Pro, Business) are unchanged — this only affects renter plans (Basic, Plus, Elite)
