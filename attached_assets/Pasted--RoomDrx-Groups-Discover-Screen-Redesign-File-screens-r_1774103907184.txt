# RoomDrx — Groups Discover Screen Redesign

## File: `screens/renter/GroupsScreen.tsx`

---

## Issues Being Fixed

1. **Avatar cluster looks unfinished** — dashed "+" empty circle next to member photos reads as broken UI
2. **Sponsored ad breaks swipe flow** — card sitting between the group card and action buttons is jarring
3. **AI button looks like a notification badge** — red circle with chip icon reads as an error indicator
4. **Card blends into background** — `#1a1a1a` on `#111` has almost no visual separation
5. **Action buttons lack visual weight** — no glow or shadow, feel flat
6. **Swipe hint labels appear bunched** — "SkipLike" reads as one word on screen
7. **Member chips feel plain** — "Film ♂" occupation tag lacks visual polish

---
