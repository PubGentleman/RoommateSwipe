Feature Parity & Paywall Gate Audit
Audit the entire app to verify that every feature listed in the subscription pricing tiers is built, gated correctly, and enforced at the backend. Do not just check UI labels — verify the actual logic.
Renter features to verify:
For Basic (free): confirm there is a hard limit of 5 interest cards per day per user, a limit of 10 listing views per day, and that only basic match filters are available (budget range, location, move-in date). Confirm these limits reset at midnight.
For Plus ($14.99/mo): confirm that interest card limits are removed entirely, that lifestyle and schedule filters are available and functional in the filter sheet, that the AI assistant button is visible and active, that a "Who Liked You" screen or section exists showing users who sent interest cards, and that rewind (undo last swipe) works up to 5 times per day and super like works up to 3 times per day.
For Elite ($29.99/mo): confirm that a weekly profile boost is scheduled automatically, that Elite profiles are ranked higher in match results, that read receipts are shown in the chat screen, that a background and income verification flow exists, and that super likes and rewinds are fully unlimited.
Host features to verify:
For Starter (free): confirm that hosts can only have 1 active listing at a time (additional listings should be blocked or paused), that inquiry responses are capped at 5 per month, and that analytics only shows view count — no match rate, inquiry count, or conversion rate.
For Pro ($29.99/mo): confirm that up to 5 listings can be active simultaneously, that there is no cap on inquiry responses, that Pro listings appear higher in Explore results than Starter listings, that the full analytics dashboard shows views, match rate, inquiry count, and conversion rate, and that a verified host badge is displayed on the host's profile and listings.
For Business ($79.99/mo): confirm that there is no listing cap, that Business listings appear above Pro listings in search, that a featured listing slot exists on the Explore screen (separate from the regular feed), that advanced analytics are available, and that a dedicated support contact or chat option exists.
One-time purchases to verify:
Confirm that Listing Boost ($4.99) sets a 7-day boost flag on the listing and moves it up in results, then expires automatically. Confirm that Host Verification Badge ($9.99) triggers an ID verification flow and sets a permanent badge on the host profile. Confirm that Super Interest ($0.99) moves the renter's interest card to the top of the host's inquiry queue with a gold badge indicator.
For each feature above: if it is not yet built, create a TODO comment in the relevant file with the label // MISSING FEATURE: followed by the feature name and which plan it belongs to. If it exists but is not gated behind the correct plan, create a comment labeled // PAYWALL MISSING: with the feature name. Return a summary list of all missing features and all ungated features found.
