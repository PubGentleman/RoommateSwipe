Call shouldRecalculate() when the AI sheet opens. Only recompute insights 
where it returns true. Show cached values instantly for all others — 
this makes the sheet feel fast and not loading every time.

Dispatch trigger events from these locations in the app:
- After any profile field is saved → dispatch 'profile_change'
- After the user leaves the swipe screen → dispatch 'swipe_session_end'  
- After preferences/filters are saved → dispatch 'filter_change'
- After a message is sent or a conversation is opened → dispatch 'message_activity'
- Every time the app opens (AppState active) → dispatch 'daily' if 24hrs have passed

Use a simple event emitter or call insightRefresh.triggerRecalculation(type) 
directly at each dispatch point.

Card tap destinations:

PROFILE COMPLETION card:
- Navigate to the specific onboarding/profile step that has the first 
  missing field

- If missing 'occupation' → navigate to the Location & Work step
- If missing 'lifestyle tags' → navigate to the Interests & Lifestyle step

MATCH POOL IMPACT card:

RESPONSE RATE card:

PROFILE TIPS card:

CHAT COACHING card:
