# Rhome — Agent Matchmaker: Complete Build Prompt

Build the **Agent Matchmaker** system for Rhome. This is a premium feature that lets real estate agents browse individual renters, use AI to find compatible matches, assemble groups, send invites, and place groups into their listings — all tracked through a pipeline with a placement fee model.

---

## Context: Why This Feature Exists

A real estate agent earns 50–100% of one month's rent per placement. On a $4,000/month 4-bedroom, that's a $2,000–4,000 commission per deal. The agent needs to find 4 compatible renters, group them, and place them. Rhome AI does the matching work in seconds instead of the agent spending days on it manually.

Because the earning potential is so high, this is a premium-priced feature with a subscription + per-placement fee model.

---

### Remove agents from the standard host pricing tiers entirely.
Agents get their own pricing structure separate from individual hosts and companies.

### Agent Plan Tiers:

```
PRO — $99.99/month
- Full AI renter suggestions per listing
- Full AI group suggestions + best group recommendation
- Shortlist up to 50 renters at once
- 5 active assembled groups at a time
- Max 10 placements per month
- $49 placement fee per successful placed group
- Compatibility matrix (color-coded grid)
- AI chat for matching queries

BUSINESS — $199.99/month
- Everything in Pro
- Unlimited shortlisting
- Unlimited active groups
- Unlimited placements per month
- $25 placement fee per successful placed group
- Priority renter visibility (their listings shown first to renters)
- Team seats (up to 5 agents under one account)
- Advanced analytics (placement rate, avg days to place, revenue tracking)

PAY PER USE — No subscription
- $149 flat fee per placement
- Manual browsing only
- No AI, no group building tools
- For occasional/part-time agents
```

### How placement fee is charged:
When an agent marks a group as "Placed" in their pipeline, Stripe automatically charges the placement fee to the payment method on file. The agent gets a receipt. This ties Rhome's revenue directly to the agent's success.

```ts
const applyFilters = () => {
  let filtered = allRenters;

  // Budget filter — renter's max budget must cover their share of listing rent
  if (selectedListing) {
    const sharePerPerson = selectedListing.rent / selectedListing.bedrooms;
    filtered = filtered.filter(r =>
      (r.profiles?.budget_max ?? 0) >= sharePerPerson
    );
  }

  // Move-in date filter — within 30 days of listing available date
  if (selectedListing?.availableDate) {
    const availDate = new Date(selectedListing.availableDate);
    filtered = filtered.filter(r => {
      if (!r.profiles?.move_in_date) return true; // flexible
      const moveIn = new Date(r.profiles.move_in_date);
      const diffDays = Math.abs((moveIn.getTime() - availDate.getTime()) / (1000 * 60 * 60 * 24));
      return diffDays <= 30;
    });
  }

  // Room type filter
  if (roomTypeFilter !== 'any') {
    filtered = filtered.filter(r => r.profiles?.room_type === roomTypeFilter);
  }

  // Neighborhood filter
  if (neighborhoodFilter) {
    filtered = filtered.filter(r => r.neighborhood === neighborhoodFilter);
  }

  setFilteredRenters(filtered);
```

### AI scoring logic (run client-side using existing utils):
```ts
const generateAISuggestions = (renters: Renter[], listing: Listing) => {
  const sharePerPerson = listing.rent / listing.bedrooms;

  return renters
    .map(renter => {
      const profile = renter.profiles;

      // Hard filters first
      const budgetOk = (profile?.budget_max ?? 0) >= sharePerPerson;
      const moveInOk = isMoveInCompatible(profile?.move_in_date, listing.availableDate);

      if (!budgetOk || !moveInOk) return null;

      // Score remaining factors
      const budgetScore = Math.min(100,
        ((profile?.budget_max ?? 0) / sharePerPerson) * 100
      );
      const profileComplete = calculateProfileCompleteness(renter);
      const lifestyleScore = 70; // base score — refined by compatibility matrix

      const listingFitScore = Math.round(
        (budgetScore * 0.4) +
        (profileComplete * 0.3) +
        (lifestyleScore * 0.3)
      );

      let reason = '';
      if (!budgetOk) reason = 'Budget too low for this listing';
      else if (!moveInOk) reason = 'Move-in date too far out';
      else if (listingFitScore >= 90) reason = 'Strong profile, move-in ready';
      else if (listingFitScore >= 75) reason = 'Good fit, verify lifestyle preferences';
      else reason = 'Marginal fit — check budget flexibility';

      return { renter, listingFitScore, reason, budgetOk, moveInOk };
    })
    .filter(Boolean)
    .sort((a, b) => b!.listingFitScore - a!.listingFitScore)
    .slice(0, 5); // top 5 suggestions
};

const isMoveInCompatible = (renterDate: string | null, listingDate: string | null): boolean => {
  if (!renterDate || !listingDate) return true; // flexible
  const moveIn = new Date(renterDate);
  const available = new Date(listingDate);
  const diffDays = Math.abs((moveIn.getTime() - available.getTime()) / (1000 * 60 * 60 * 24));
  return diffDays <= 30;
};
```

### Best group suggestion:
```ts
const generateBestGroupSuggestion = (shortlisted: Renter[], listing: Listing) => {
  if (shortlisted.length < 2) return null;

  const needed = listing.bedrooms;
  const combinations = getCombinations(shortlisted, needed); // all possible groups of size `needed`

  const scored = combinations.map(group => {
    // Check combined budget covers rent
    const combinedMin = group.reduce((sum, r) => sum + (r.profiles?.budget_min ?? 0), 0);
    const combinedMax = group.reduce((sum, r) => sum + (r.profiles?.budget_max ?? 0), 0);
    const coversRent = combinedMax >= listing.rent;

    // Average compatibility between all pairs
    const pairs = getPairs(group);
    const avgCompatibility = pairs.length > 0
      ? Math.round(pairs.reduce((sum, [a, b]) =>
          sum + calculateCompatibility(a, b), 0) / pairs.length)
      : 0;

    return {
      group,
      names: group.map(r => r.full_name),
      avgCompatibility,
      combinedBudgetMin: combinedMin,
      combinedBudgetMax: combinedMax,
      coversRent,
      score: coversRent ? avgCompatibility : avgCompatibility * 0.5,
    };
  });

  return scored.sort((a, b) => b.score - a.score)[0] ?? null;
};
```

```ts
const acceptAgentInvite = async (inviteId: string, groupId: string) => {
  // 1. Update invite status
  await supabase.from('agent_group_invites')
    .update({ status: 'accepted', responded_at: new Date().toISOString() })
    .eq('id', inviteId);

  // 2. Add renter to group
  await supabase.from('group_members').insert({
    group_id: groupId,
    user_id: user.id,
    role: 'member',
  });

  // 3. Notify agent
  await sendPushNotification(agentId, {
    title: 'Renter accepted your invite',
    body: `${user.fullName} accepted the invite to your group for ${listingTitle}`,

  // 4. Check if all renters accepted → update group to 'active'
  const { count: pendingCount } = await supabase
    .from('agent_group_invites')
    .select('*', { count: 'exact', head: true })
    .eq('group_id', groupId)
    .eq('status', 'pending');

  if (pendingCount === 0) {
    await supabase.from('groups')
      .update({ group_status: 'active' })
      .eq('id', groupId);
```

```ts
    body: `${user.fullName} declined. You can invite another renter from your shortlist.`,
```

```ts
async function buildAgentSystemPrompt(
  supabase: any,
  userId: string,
  agentPlan: string
): Promise<string> {
  // Get agent's listings
  const { data: listings } = await supabase
    .from('listings')
    .select('id, title, rent, bedrooms, neighborhood, status')
    .eq('host_id', userId)
    .eq('is_active', true);

  // Get shortlisted renter count
  const { count: shortlistCount } = await supabase
    .from('agent_shortlists')
    .select('*', { count: 'exact', head: true })
    .eq('agent_id', userId);

  // Get active group count
  const { count: activeGroups } = await supabase
    .from('groups')
    .select('*', { count: 'exact', head: true })
    .eq('created_by_agent', userId)
    .in('group_status', ['invited', 'active']);

  const listingSummary = listings?.map((l: any) =>
    `"${l.title}" — $${l.rent}/mo, ${l.bedrooms}BR in ${l.neighborhood}`
  ).join('; ') ?? 'no active listings';

  return `You are Rhome AI, helping a real estate agent find and match renters for their listings.

AGENT PLAN: ${agentPlan}
ACTIVE LISTINGS: ${listingSummary}
SHORTLISTED RENTERS: ${shortlistCount ?? 0}
ACTIVE ASSEMBLED GROUPS: ${activeGroups ?? 0}

Your job is to help this agent:
- Find renters who fit their listing requirements (right budget, right move-in date, right lifestyle)
- Assess whether two or more renters would be compatible roommates
- Suggest which renters from their shortlist to group together
- Help craft compelling invite messages to send to renters
- Manage their group pipeline efficiently

KEY RULES:
- Budget is a HARD filter. A renter's max budget must cover their share of the rent (listing rent ÷ bedrooms). Never suggest someone who can't afford their share.
- Move-in date is a HARD filter. Renter's move-in must be within 30 days of listing availability. Flag anyone outside this window.
- After hard filters pass, rank by: lifestyle compatibility, profile completeness, neighborhood match.
- When asked to compare renters, give a direct recommendation — don't be vague.
- Never suggest roommate matching content — this agent is placing tenants, not finding a roommate for themselves.
- Keep responses conversational and direct. Agents are busy professionals.`;
}
```

## Part 12 — Stripe Placement Fee Edge Function

Create `supabase/functions/charge-placement-fee/index.ts`:

```ts
import Stripe from 'https://esm.sh/stripe@12.0.0?target=deno';
import { createClient } from 'https://esm.sh/@supabase/supabase-js@2';

const stripe = new Stripe(Deno.env.get('STRIPE_SECRET_KEY')!);
const supabase = createClient(
  Deno.env.get('SUPABASE_URL')!,
  Deno.env.get('SUPABASE_SERVICE_ROLE_KEY')!
);

Deno.serve(async (req) => {
  const { agentId, placementId, amountCents } = await req.json();

  // Get agent's Stripe customer ID
  const { data: sub } = await supabase
    .from('subscriptions')
    .select('stripe_customer_id')
    .eq('user_id', agentId)
    .single();

  if (!sub?.stripe_customer_id) {
    return new Response(JSON.stringify({ error: 'No payment method on file' }), { status: 400 });
  }

  // Charge the placement fee
  const paymentIntent = await stripe.paymentIntents.create({
    amount: amountCents,
    currency: 'usd',
    customer: sub.stripe_customer_id,
    confirm: true,
    off_session: true,
    description: 'Rhome placement fee',
    metadata: { agentId, placementId },
  });

  // Update placement billing status
  await supabase.from('agent_placements')
    .update({
      stripe_payment_intent_id: paymentIntent.id,
      billing_status: paymentIntent.status === 'succeeded' ? 'charged' : 'failed',
    })
    .eq('id', placementId);

  return new Response(JSON.stringify({ success: true, status: paymentIntent.status }));
});
```
