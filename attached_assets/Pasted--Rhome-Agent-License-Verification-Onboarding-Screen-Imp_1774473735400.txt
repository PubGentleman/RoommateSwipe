# Rhome — Agent License Verification & Onboarding Screen Improvements
## Replit Agent Prompt

---

## Overview

This prompt covers three improvements to the agent onboarding flow:

1. **License upload** — agents can upload a photo/PDF of their license in addition to entering their license number
2. **ARELLO API verification** — automatically cross-reference license number + name against state licensing board records
3. **Benefit callouts** — fill empty screen space on all host onboarding screens with feature highlights

---

## Part 1 — Database Migration

Create `034_agent_verification.sql` and run in Supabase SQL Editor:

```sql
ALTER TABLE public.users
  ADD COLUMN IF NOT EXISTS license_state TEXT,
  ADD COLUMN IF NOT EXISTS license_document_url TEXT,
  ADD COLUMN IF NOT EXISTS license_verified BOOLEAN DEFAULT false,
  ADD COLUMN IF NOT EXISTS license_verified_at TIMESTAMPTZ,
  ADD COLUMN IF NOT EXISTS license_verification_status TEXT DEFAULT 'unverified'
    CHECK (license_verification_status IN ('unverified', 'pending', 'verified', 'failed', 'manual_review'));
```

### 2e — Updated handleContinue

```typescript
async function handleContinue() {
  if (!canContinue) return;
  setSaving(true);
  try {
    // Upload document if selected
    let documentUrl: string | undefined;
    if (licenseDocumentUri) {
      documentUrl = await uploadLicenseDocument(licenseDocumentUri);
    }

    // Save fields
    await updateUser({
      hostType: 'agent',
      licenseNumber: values.licenseNumber?.trim(),
      licenseState: values.licenseState,
      agencyName: values.agencyName?.trim() || undefined,
      hostTypeLockedAt: new Date().toISOString(),
    });

    // Run ARELLO verification via Edge Function
    setVerifying(true);
    const verified = await verifyAgentLicense({
      licenseNumber: values.licenseNumber?.trim(),
      licenseState: values.licenseState,
      firstName: user?.firstName,
      lastName: user?.lastName,
    });

    if (verified) {
      await updateUser({ licenseVerificationStatus: 'verified' });
    } else {
      await updateUser({ licenseVerificationStatus: 'manual_review' });
    }

    setVerifying(false);

    // Proceed to subscription regardless of verification status
    if (isFromSettings) {
      navigation.replace('HostSubscription');
    } else {
      await completeOnboardingStep('plan');
    }
  } catch (e) {
    console.error(e);
  } finally {
    setSaving(false);
  }
}
```

## Part 3 — Supabase Edge Function: verifyAgentLicense

Create `supabase/functions/verify-agent-license/index.ts`:

```typescript
import { serve } from 'https://deno.land/std@0.177.0/http/server.ts';

serve(async (req) => {
  const { licenseNumber, licenseState, firstName, lastName } = await req.json();

  try {
    // Call ARELLO Licensee Lookup API
    const arelloResponse = await fetch(
      `https://api.arello.com/licensee/lookup`,
      {
        method: 'POST',
        headers: {
          'Content-Type': 'application/json',
          'Authorization': `Bearer ${Deno.env.get('ARELLO_API_KEY')}`,
        },
        body: JSON.stringify({
          licenseNumber,
          state: licenseState,
          firstName,
          lastName,
        }),
      }
    );

    const data = await arelloResponse.json();

    // Check if license matches
    const isVerified = data.status === 'active' &&
      data.licenseNumber === licenseNumber &&
      data.state === licenseState;

    // Update user's verification status in Supabase
    const supabaseUrl = Deno.env.get('SUPABASE_URL')!;
    const supabaseKey = Deno.env.get('SUPABASE_SERVICE_ROLE_KEY')!;

    await fetch(`${supabaseUrl}/rest/v1/users?id=eq.${req.headers.get('x-user-id')}`, {
      method: 'PATCH',
      headers: {
        'apikey': supabaseKey,
        'Authorization': `Bearer ${supabaseKey}`,
        'Content-Type': 'application/json',
      },
      body: JSON.stringify({
        license_verified: isVerified,
        license_verified_at: isVerified ? new Date().toISOString() : null,
        license_verification_status: isVerified ? 'verified' : 'manual_review',
      }),
    });

    return new Response(JSON.stringify({ verified: isVerified }), {
      headers: { 'Content-Type': 'application/json' },
    });
  } catch (err) {
    return new Response(JSON.stringify({ verified: false, error: err.message }), {
      status: 500,
      headers: { 'Content-Type': 'application/json' },
    });
  }
});
```

Add `ARELLO_API_KEY` to Supabase Edge Function secrets.

### Client-side call:

```typescript
const verifyAgentLicense = async (params: {
  licenseNumber: string;
  licenseState: string;
  firstName?: string;
  lastName?: string;
}): Promise<boolean> => {
  const { data, error } = await supabase.functions.invoke('verify-agent-license', {
    body: params,
    headers: { 'x-user-id': user?.id },
  });
  if (error) return false;
  return data.verified === true;
};
```
