## What's being added

A **Listing Type** filter that lets renters quickly switch between:
- **Any** — show all listings (default)
- **Private Room** — room in a shared apartment (already partially exists as a chip)
- **Entire Unit** — full apartment/house to themselves
- **Sublet** — short-term sublets

Added in 3 places: filter chips row, Filters modal, and the listing filter logic.

## Place 1 — Filter Chips Row: Replace "Private Room" chip with type chips

The current "Private Room" chip is a one-off that doesn't cover "Entire Unit". Replace it with a proper set of type chips that are mutually exclusive (selecting one deselects the others).

### Add state:

```ts
// In ExploreScreen — add listing type filter state:
const [listingTypeFilter, setListingTypeFilter] = useState<'any' | 'room' | 'entire' | 'sublet'>('any');

// Handler — toggle: tapping the active type resets to 'any'
const handleListingTypeChip = (type: 'room' | 'entire' | 'sublet') => {
  setListingTypeFilter(prev => prev === type ? 'any' : type);
};
```

## Place 3 — Filter Logic: Wire Up to Listings

In `ExploreScreen`, update the `filteredListings` memo to apply the listing type filter.

```ts
// Map filter key → listing field values
// Check how your Supabase listings table stores type:
// Common field names: listing.type, listing.listing_type, listing.room_type
// Common values: 'room' | 'entire_unit' | 'sublet'  OR  'Private Room' | 'Entire Unit' | 'Sublet'

const TYPE_MAP: Record<string, string[]> = {
  room:   ['room', 'private_room', 'Private Room'],
  entire: ['entire', 'entire_unit', 'Entire Unit'],
  sublet: ['sublet', 'Sublet'],
};

const filteredListings = useMemo(() => {
  let results = allListings;

  // City filter
  results = results.filter(l => l.city === selectedCity);

  // Neighborhood filter
  if (selectedNeighborhood) {
    results = results.filter(l =>
      l.neighborhood?.toLowerCase().includes(selectedNeighborhood.toLowerCase())
    );
  }

  // ── NEW: Listing type filter ──────────────────
  if (listingTypeFilter !== 'any') {
    const allowedValues = TYPE_MAP[listingTypeFilter] ?? [];
    results = results.filter(l =>
      allowedValues.some(v =>
        l.type?.toLowerCase() === v.toLowerCase() ||
        l.listing_type?.toLowerCase() === v.toLowerCase() ||
        l.room_type?.toLowerCase() === v.toLowerCase()
      )
    );
  }
  // ─────────────────────────────────────────────

  // Best Match sort
  if (activeFilter === 'best_match') {
    results = [...results].sort((a, b) => (b.matchScore ?? 0) - (a.matchScore ?? 0));
  }

  // Budget filter
  if (activeFilter === 'under_2k') {
    results = results.filter(l => l.price <= 2000);
  }

  // Pet friendly filter
  if (activeFilter === 'pet_friendly') {
    results = results.filter(l => l.petFriendly);
  }

  return results;
}, [
  allListings,
  selectedCity,
  selectedNeighborhood,
  listingTypeFilter,   // ← add to dependency array
  activeFilter,
  budgetMin,
  budgetMax,
  // ...other filter deps
]);
```
