**PART 3 — VALIDATION RULES**

The **"Next →"** button unlocks only when ALL of these are true:
- At least 1 tag selected from **Lifestyle**
- At least 1 tag selected from **Habits**
- At least 1 tag selected from **Hobbies**
- At least **3 tags total** across all 5 categories combined

Social and Diet have no minimum — they are fully optional.

**Validation function — add to `utils/profileReminderUtils.ts`:**

```ts
export const validateInterestTags = (
  selectedTags: string[],
  tagsByCategory: Record<string, string[]>
): { valid: boolean; message?: string } => {
  const REQUIRED_CATEGORIES = ['lifestyle', 'habits', 'hobbies'];

  for (const category of REQUIRED_CATEGORIES) {
    const categoryTags = tagsByCategory[category] ?? [];
    const hasSelection = selectedTags.some(tag => categoryTags.includes(tag));
    if (!hasSelection) {
      return {
        valid: false,
        message: `Add at least 1 ${category.charAt(0).toUpperCase() + category.slice(1)} tag to continue`
      };
    }
  }

  if (selectedTags.length < 3) {
    return {
      valid: false,
      message: 'Pick 1 more tag from any category to continue'
    };
  }

  return { valid: true };
};
```
