## Change 2 — Remove the Redundant Second Location Row

The screen currently has **two** location indicators:
1. The one in the header row (being upgraded above)
2. A second `"📍 New York  ▾  19 listings"` row beneath the header

The second row is now redundant — the upgraded location picker in the header shows city + neighborhood. **Delete the second location row entirely.**

```tsx
// REMOVE this entire View:
```

## Change 3 — Neighborhood Location Picker Bottom Sheet

Add a new bottom sheet that opens when the location picker is tapped. It shows city selection at the top and neighborhood chips below.

```tsx
// Add state:
const [showLocationSheet, setShowLocationSheet] = useState(false);
const [selectedCity, setSelectedCity] = useState('New York');
const [selectedNeighborhood, setSelectedNeighborhood] = useState<string | null>(null);

// Neighborhood data per city (expand as needed):
const NEIGHBORHOODS: Record<string, string[]> = {
  'New York': [
    'All Neighborhoods',
    'Upper East Side', 'Upper West Side', 'Midtown',
    'Chelsea', 'Hell\'s Kitchen', 'Greenwich Village',
    'East Village', 'Lower East Side', 'SoHo', 'Tribeca',
    'Financial District', 'Brooklyn Heights', 'Williamsburg',
    'Bushwick', 'Astoria', 'Long Island City', 'Harlem',
    'Washington Heights', 'Bronx',
  ],
  'Los Angeles': [
    'All Neighborhoods',
    'Downtown LA', 'Hollywood', 'Silver Lake', 'Echo Park',
    'Los Feliz', 'Koreatown', 'Mid-City', 'Culver City',
    'Santa Monica', 'Venice', 'West Hollywood', 'Brentwood',
  ],
  // add more cities...
};

// Bottom sheet JSX:
<Modal
  visible={showLocationSheet}
  animationType="slide"
  transparent
  onRequestClose={() => setShowLocationSheet(false)}
>
  <Pressable style={styles.locSheetOverlay} onPress={() => setShowLocationSheet(false)} />
  <View style={styles.locSheet}>
    {/* Handle */}
    <View style={styles.locSheetHandle} />

    {/* Title */}
    <Text style={styles.locSheetTitle}>Choose Location</Text>

    {/* City selector */}
    <Text style={styles.locSheetSectionLabel}>City</Text>
    <ScrollView
      horizontal
      showsHorizontalScrollIndicator={false}
      style={{ overflow: 'visible' }}
      contentContainerStyle={styles.locCityScroll}
    >
      {Object.keys(NEIGHBORHOODS).map((city) => (
        <Pressable
          key={city}
          style={[styles.locCityChip, selectedCity === city && styles.locCityChipActive]}
          onPress={() => { setSelectedCity(city); setSelectedNeighborhood(null); }}
        >
          <Text style={[styles.locCityText, selectedCity === city && styles.locCityTextActive]}>
            {city}
          </Text>
        </Pressable>
      ))}
    </ScrollView>

    {/* Neighborhood grid */}
    <Text style={styles.locSheetSectionLabel}>Neighborhood</Text>
    <ScrollView
      showsVerticalScrollIndicator={false}
      contentContainerStyle={styles.locNeighborhoodGrid}
    >
      {(NEIGHBORHOODS[selectedCity] ?? []).map((hood) => {
        const isAll = hood === 'All Neighborhoods';
        const isActive = isAll
          ? selectedNeighborhood === null
          : selectedNeighborhood === hood;
        return (
          <Pressable
            key={hood}
            style={[
              styles.locHoodChip,
              isActive && styles.locHoodChipActive,
              isAll && styles.locHoodChipAll,
            ]}
            onPress={() => {
              setSelectedNeighborhood(isAll ? null : hood);
              setShowLocationSheet(false);
            }}
          >
            <Text style={[styles.locHoodText, isActive && styles.locHoodTextActive]}>
              {hood}
            </Text>
          </Pressable>
        );
      })}
    </ScrollView>
  </View>
</Modal>
```

## Change 4 — Wire Neighborhood Filter into Listings

After the user picks a neighborhood, filter the listings list:

```ts
// In your listings fetch / filter logic:
const filteredListings = useMemo(() => {
  let results = allListings;

  // City filter (already existing)
  results = results.filter(l => l.city === selectedCity);

  // Neighborhood filter — NEW
  if (selectedNeighborhood) {
    results = results.filter(l =>
      l.neighborhood?.toLowerCase().includes(selectedNeighborhood.toLowerCase())
    );
  }

  // ...existing filters (price, type, etc.)
  return results;
}, [allListings, selectedCity, selectedNeighborhood, /* other filters */]);
```

Also update the listing count display to reflect neighborhood filtering:

```ts
// Dynamic count label:
const locationLabel = selectedNeighborhood ?? selectedCity;
const countLabel = `${filteredListings.length} listing${filteredListings.length !== 1 ? 's' : ''}`;
// Shows as subtitle in the location picker: "19 listings" or "Financial District"
```
