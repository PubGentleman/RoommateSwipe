# RoomDrx — Chat / Messages Screen Redesign

## File: `screens/shared/ChatListScreen.tsx`
(may also be named `MessagesScreen.tsx` or `InboxScreen.tsx`)

---

## Issues Being Fixed

1. **"Unmatched" badge is the first thing you see** — dominates both rows, looks like an error state
2. **"Direct" chip is purple** — breaks the coral accent system used everywhere else
3. **Conversation rows are cramped** — name, badges, match %, and timestamp all crammed on one line
4. **No filter tabs** — no way to separate Matches / Groups / Direct messages
5. **Header is plain** — compose button is a grey circle with no visual weight; AI button has no label
6. **Online dot is tiny** — easy to miss, not clearly indicating active status
7. **Massive empty space** — bottom 80% of screen is dead black
8. **No unread count on individual rows** — the badge shows 2 unread but it's small and in an odd position

---

## Change 3 — Conversation Rows: Full Redesign

Replace the cramped single-line layout with a clean 3-row card structure.

```tsx
const renderConversation = ({ item }: { item: Conversation }) => {
  const isUnmatched = item.status === 'unmatched';
  const isOnline = item.isOnline;

  return (
    <Pressable
      style={[styles.convRow, isUnmatched && styles.convRowUnmatched]}
      onPress={() => !isUnmatched && navigation.navigate('Chat', { id: item.id })}
    >
      {/* Avatar */}
      <View style={styles.convAvatarWrap}>
        {item.photo ? (
          <Image source={{ uri: item.photo }} style={[styles.convAvatar, isUnmatched && { opacity: 0.4 }]} />
        ) : (
          <LinearGradient
            colors={['#667eea', '#764ba2']}
            style={[styles.convAvatar, isUnmatched && { opacity: 0.4 }]}
          >
            <Text style={styles.convAvatarLetter}>{item.name[0]}</Text>
          </LinearGradient>
        )}
        {/* Online dot */}
        {isOnline && !isUnmatched && (
          <View style={styles.onlineDot} />
        )}
      </View>

      {/* Content */}
      <View style={styles.convContent}>
        {/* Row 1: Name + timestamp */}
        <View style={styles.convTopRow}>
          <Text style={[styles.convName, isUnmatched && styles.convNameMuted]} numberOfLines={1}>
            {item.name}
          </Text>
          <Text style={[styles.convTime, item.unreadCount > 0 && styles.convTimeUnread]}>
            {item.timestamp}
          </Text>
        </View>

        {/* Row 2: Message preview + unread badge */}
        <View style={styles.convMidRow}>
          <Text
            style={[styles.convPreview, isUnmatched && styles.convPreviewMuted]}
            numberOfLines={1}
          >
            {isUnmatched ? 'This match has ended' : item.lastMessage}
          </Text>
          {item.unreadCount > 0 && !isUnmatched && (
            <View style={styles.unreadBadge}>
              <Text style={styles.unreadBadgeText}>{item.unreadCount}</Text>
            </View>
          )}
        </View>

        {/* Row 3: Tags (only shown when NOT unmatched) */}
        {!isUnmatched && (
          <View style={styles.convTagRow}>
            {item.matchScore && (
              <View style={styles.matchTag}>
                <Feather name="heart" size={9} color="#ff6b5b" />
                <Text style={styles.matchTagText}>{item.matchScore}% match</Text>
              </View>
            )}
            {item.type === 'direct' && (
              <View style={styles.directTag}>
                <Feather name="send" size={9} color="rgba(255,255,255,0.4)" />
                <Text style={styles.directTagText}>Direct</Text>
              </View>
            )}
            {item.type === 'group' && (
              <View style={styles.groupTag}>
                <Feather name="users" size={9} color="rgba(255,255,255,0.4)" />
                <Text style={styles.groupTagText}>Group</Text>
              </View>
            )}
          </View>
        )}

        {/* Unmatched state: subtle archive prompt */}
        {isUnmatched && (
          <Text style={styles.unmatchedHint}>Tap to archive</Text>
        )}
      </View>
    </Pressable>
  );
};
```
