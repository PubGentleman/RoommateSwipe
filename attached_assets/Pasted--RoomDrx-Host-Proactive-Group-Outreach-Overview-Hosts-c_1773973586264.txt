# RoomDrx — Host Proactive Group Outreach

## Overview

Hosts can browse discoverable renter groups actively looking for a place and send them a message about an available listing. This is **pre-rental proactive outreach** — different from post-rental outreach (which is for after a listing is rented).

Only groups with `discoverable = true` appear here. Robust spam controls protect groups from being flooded. When a host hits their daily limit they can **pay to unlock more sends** for that day.

---

## Plan Limits

| Plan | Messages / Month | Daily Cap | Hourly Cap | Per-Group Cooldown |
|------|-----------------|-----------|------------|-------------------|
| Starter | 0 | — | — | — |
| Pro | 5 | 3/day | 2/hour | 30 days |
| Business | Unlimited | **10/day** | **3/hour** | 30 days |

### Pay-to-Unlock (after hitting daily cap)

When a host hits their daily cap, they can purchase additional sends for that day:

| Extra Sends | Price |
|-------------|-------|
| +3 messages today | $4.99 |
| +10 messages today | $12.99 |

These expire at midnight — unused paid sends do not roll over.

---

## Spam Protection Rules (All Plans)

| Rule | Enforcement |
|------|------------|
| Daily send cap | Tracked in `host_outreach_quota` per day |
| Hourly rate limit | Max N messages per rolling 60-minute window |
| Per-group cooldown | 30 days before same host can message same group again |
| One message per group per day | Cannot send twice in 24h even with paid unlocks |
| Minimum message length | 50 characters — prevents "hi" spam |
| Report threshold | 3 group reports = outreach suspended pending review |
| No duplicate content detection | Same message copy-pasted to 5+ groups in 1 hour = flagged |

## Database Changes

### 1. `host_group_outreach` table (pre-rental, separate from post-rental `host_outreach`)

```sql
CREATE TABLE host_group_outreach (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  host_id UUID REFERENCES auth.users(id) ON DELETE CASCADE,
  group_id UUID REFERENCES groups(id) ON DELETE CASCADE,
  listing_id UUID REFERENCES listings(id) ON DELETE SET NULL, -- optional: which listing they're promoting
  message TEXT NOT NULL,
  sent_at TIMESTAMPTZ DEFAULT NOW(),
  reported BOOLEAN DEFAULT false,
  report_count INT DEFAULT 0,
  report_reason TEXT,
  payment_intent_id TEXT -- set if sent via paid unlock
);

-- Host inserts own outreach
CREATE POLICY "Hosts send proactive outreach"
ON host_group_outreach FOR INSERT
WITH CHECK (auth.uid() = host_id);

-- Host reads own sent messages
CREATE POLICY "Hosts read own outreach"
ON host_group_outreach FOR SELECT
USING (auth.uid() = host_id);

-- Group members read outreach sent to their group
CREATE POLICY "Group members read outreach"
ON host_group_outreach FOR SELECT
USING (
  group_id IN (
    SELECT group_id FROM group_members WHERE user_id = auth.uid()
  )
);

-- Group members can report
CREATE POLICY "Group members report outreach"
ON host_group_outreach FOR UPDATE
USING (
  group_id IN (
    SELECT group_id FROM group_members WHERE user_id = auth.uid()
  )
)
WITH CHECK (report_count = host_group_outreach.report_count + 1);
```

### 2. `host_outreach_daily_quota` table

Tracks daily + hourly usage with paid unlock credits.

```sql
CREATE TABLE host_outreach_daily_quota (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  host_id UUID REFERENCES auth.users(id) ON DELETE CASCADE,
  date_key TEXT NOT NULL,       -- "2025-03-19"
  used INT DEFAULT 0,
  paid_extra INT DEFAULT 0,     -- additional sends purchased for today
  UNIQUE(host_id, date_key)
);

CREATE POLICY "Hosts manage own daily quota"
ON host_outreach_daily_quota FOR ALL
USING (auth.uid() = host_id)
WITH CHECK (auth.uid() = host_id);
```

### 3. `host_outreach_suspended` table

Tracks hosts whose outreach is suspended due to reports.

```sql
CREATE TABLE host_outreach_suspended (
  host_id UUID PRIMARY KEY REFERENCES auth.users(id) ON DELETE CASCADE,
  reason TEXT,
  suspended_at TIMESTAMPTZ DEFAULT NOW(),
  reviewed BOOLEAN DEFAULT false
);
```

### 4. Supabase RPC for quota management

```sql
-- Increment daily usage
CREATE OR REPLACE FUNCTION increment_daily_outreach(
  p_host_id UUID,
  p_date_key TEXT
)
RETURNS VOID AS $$
BEGIN
  INSERT INTO host_outreach_daily_quota (host_id, date_key, used)
  VALUES (p_host_id, p_date_key, 1)
  ON CONFLICT (host_id, date_key)
  DO UPDATE SET used = host_outreach_daily_quota.used + 1;
END;
$$ LANGUAGE plpgsql SECURITY DEFINER;

-- Add paid unlock credits
CREATE OR REPLACE FUNCTION add_paid_outreach_credits(
  p_host_id UUID,
  p_date_key TEXT,
  p_credits INT
)
RETURNS VOID AS $$
BEGIN
  INSERT INTO host_outreach_daily_quota (host_id, date_key, paid_extra)
  VALUES (p_host_id, p_date_key, p_credits)
  ON CONFLICT (host_id, date_key)
  DO UPDATE SET paid_extra = host_outreach_daily_quota.paid_extra + p_credits;
END;
$$ LANGUAGE plpgsql SECURITY DEFINER;

-- Increment report count + auto-suspend at threshold
CREATE OR REPLACE FUNCTION report_host_outreach(
  p_outreach_id UUID,
  p_reason TEXT
)
RETURNS VOID AS $$
DECLARE
  v_host_id UUID;
  v_report_count INT;
BEGIN
  UPDATE host_group_outreach
  SET report_count = report_count + 1,
      report_reason = p_reason,
      reported = true
  WHERE id = p_outreach_id
  RETURNING host_id, report_count INTO v_host_id, v_report_count;

  -- Auto-suspend at 3 reports
  IF v_report_count >= 3 THEN
    INSERT INTO host_outreach_suspended (host_id, reason)
    VALUES (v_host_id, 'Auto-suspended: 3+ reports')
    ON CONFLICT (host_id) DO NOTHING;
  END IF;
END;
$$ LANGUAGE plpgsql SECURITY DEFINER;
```

## Supabase Edge Functions

### `create-proactive-outreach-payment`

Handles pay-to-unlock purchases when a host hits their daily limit.

```ts
// supabase/functions/create-proactive-outreach-payment/index.ts
import { serve } from 'https://deno.land/std@0.168.0/http/server.ts';
import Stripe from 'https://esm.sh/stripe@13.3.0?target=deno';
import { createClient } from 'https://esm.sh/@supabase/supabase-js@2';

const stripe = new Stripe(Deno.env.get('STRIPE_SECRET_KEY')!, { apiVersion: '2023-10-16' });

const UNLOCK_PACKAGES = {
  small: { credits: 3,  cents: 499  },
  large: { credits: 10, cents: 1299 },
};

serve(async (req) => {
  const { packageId } = await req.json();
  const authHeader = req.headers.get('Authorization')!;

  const supabase = createClient(
    Deno.env.get('SUPABASE_URL')!,
    Deno.env.get('SUPABASE_ANON_KEY')!,
    { global: { headers: { Authorization: authHeader } } }
  );

  const { data: { user } } = await supabase.auth.getUser();
  if (!user) return new Response('Unauthorized', { status: 401 });

  const pkg = UNLOCK_PACKAGES[packageId as keyof typeof UNLOCK_PACKAGES];
  if (!pkg) return new Response('Invalid package', { status: 400 });

  // Check host is not suspended
  const { data: suspended } = await supabase
    .from('host_outreach_suspended')
    .select('host_id')
    .eq('host_id', user.id)
    .maybeSingle();

  if (suspended) return new Response('Outreach suspended', { status: 403 });

  // Get or create Stripe customer
  const { data: profile } = await supabase
    .from('profiles')
    .select('stripe_customer_id, full_name')
    .eq('id', user.id)
    .single();

  let customerId = profile?.stripe_customer_id;
  if (!customerId) {
    const customer = await stripe.customers.create({
      metadata: { supabase_uid: user.id },
    });
    customerId = customer.id;
    await supabase.from('profiles').update({ stripe_customer_id: customerId }).eq('id', user.id);
  }

  const paymentIntent = await stripe.paymentIntents.create({
    amount: pkg.cents,
    currency: 'usd',
    customer: customerId,
    automatic_payment_methods: { enabled: true },
    metadata: {
      host_id: user.id,
      package_id: packageId,
      credits: pkg.credits.toString(),
      date_key: new Date().toISOString().slice(0, 10),
    },
  });

  return new Response(
    JSON.stringify({ clientSecret: paymentIntent.client_secret, credits: pkg.credits, cents: pkg.cents }),
    { headers: { 'Content-Type': 'application/json' } }
  );
});
```

## Service Layer: `hostOutreachService.ts`

```ts
import { supabase } from '../lib/supabase';
import { OUTREACH_PLAN_LIMITS } from '../constants/planLimits';

export interface OutreachQuotaStatus {
  canSend: boolean;
  used: number;
  dailyCap: number;
  paidExtra: number;
  remaining: number;
  hitDailyLimit: boolean;
  suspended: boolean;
  reason?: string; // 'SUSPENDED' | 'NO_PLAN' | 'HOURLY_LIMIT' | 'OK'
}

/**
 * Get the host's current outreach quota status for today.
 */
export async function getOutreachQuotaStatus(
  hostId: string,
  plan: string
): Promise<OutreachQuotaStatus> {
  const limit = OUTREACH_PLAN_LIMITS[plan];

  // Check if suspended
  const { data: suspended } = await supabase
    .from('host_outreach_suspended')
    .select('host_id')
    .eq('host_id', hostId)
    .maybeSingle();

  if (suspended) {
    return { canSend: false, used: 0, dailyCap: 0, paidExtra: 0, remaining: 0, hitDailyLimit: true, suspended: true, reason: 'SUSPENDED' };
  }

  if (!limit || limit.dailyCap === 0) {
    return { canSend: false, used: 0, dailyCap: 0, paidExtra: 0, remaining: 0, hitDailyLimit: true, suspended: false, reason: 'NO_PLAN' };
  }

  const dateKey = new Date().toISOString().slice(0, 10);
  const { data: quota } = await supabase
    .from('host_outreach_daily_quota')
    .select('used, paid_extra')
    .eq('host_id', hostId)
    .eq('date_key', dateKey)
    .maybeSingle();

  const used = quota?.used ?? 0;
  const paidExtra = quota?.paid_extra ?? 0;
  const totalAllowed = limit.dailyCap + paidExtra;
  const remaining = Math.max(0, totalAllowed - used);
  const hitDailyLimit = remaining <= 0;

  return {
    canSend: !hitDailyLimit,
    used,
    dailyCap: limit.dailyCap,
    paidExtra,
    remaining,
    hitDailyLimit,
    suspended: false,
    reason: hitDailyLimit ? 'DAILY_LIMIT' : 'OK',
  };
}

/**
 * Check hourly rate limit — max N messages in the last 60 minutes.
 */
export async function checkHourlyLimit(hostId: string, plan: string): Promise<boolean> {
  const limit = OUTREACH_PLAN_LIMITS[plan];
  if (!limit) return false;

  const oneHourAgo = new Date(Date.now() - 60 * 60 * 1000).toISOString();
  const { count } = await supabase
    .from('host_group_outreach')
    .select('id', { count: 'exact', head: true })
    .eq('host_id', hostId)
    .gte('sent_at', oneHourAgo);

  return (count ?? 0) < limit.hourlyCap;
}

/**
 * Check if host already messaged this group within the cooldown window (30 days).
 */
export async function checkGroupCooldown(hostId: string, groupId: string): Promise<boolean> {
  const thirtyDaysAgo = new Date(Date.now() - 30 * 24 * 60 * 60 * 1000).toISOString();
  const { data } = await supabase
    .from('host_group_outreach')
    .select('id')
    .eq('host_id', hostId)
    .eq('group_id', groupId)
    .gte('sent_at', thirtyDaysAgo)
    .maybeSingle();

  return !data; // true = can send, false = in cooldown
}

/**
 * Send a proactive outreach message to a group.
 * Enforces all spam controls before inserting.
 */
export async function sendProactiveOutreach(
  groupId: string,
  message: string,
  plan: string,
  listingId?: string
): Promise<void> {
  const user = (await supabase.auth.getUser()).data.user;
  if (!user) throw new Error('Not authenticated');

  // 1. Minimum message length
  if (message.trim().length < 50) throw new Error('MESSAGE_TOO_SHORT');

  // 2. Quota status
  const quota = await getOutreachQuotaStatus(user.id, plan);
  if (quota.suspended) throw new Error('SUSPENDED');
  if (!quota.canSend) throw new Error('DAILY_LIMIT_REACHED');

  // 3. Hourly rate limit
  const hourlyOk = await checkHourlyLimit(user.id, plan);
  if (!hourlyOk) throw new Error('HOURLY_LIMIT_REACHED');

  // 4. Per-group cooldown
  const cooldownOk = await checkGroupCooldown(user.id, groupId);
  if (!cooldownOk) throw new Error('GROUP_COOLDOWN');

  // 5. Insert message
  const { error } = await supabase.from('host_group_outreach').insert({
    host_id: user.id,
    group_id: groupId,
    listing_id: listingId ?? null,
    message: message.trim(),
  });

  if (error) throw error;

  // 6. Increment daily quota
  const dateKey = new Date().toISOString().slice(0, 10);
  await supabase.rpc('increment_daily_outreach', {
    p_host_id: user.id,
    p_date_key: dateKey,
  });

  // 7. Notify group members
  await supabase.functions.invoke('send-push-notification', {
    body: {
      groupId,
      type: 'host_proactive_outreach',
      title: '🏠 A host wants to reach your group',
      body: message.trim().slice(0, 80) + (message.length > 80 ? '...' : ''),
      data: { type: 'host_proactive_outreach', groupId },
    },
  });
}

/**
 * Get discoverable groups for the host to browse (excludes already-messaged in 30 days).
 */
export async function getDiscoverableGroups(
  city: string,
  hostId: string
): Promise<any[]> {
  // Groups messaged in last 30 days (in cooldown)
  const thirtyDaysAgo = new Date(Date.now() - 30 * 24 * 60 * 60 * 1000).toISOString();
  const { data: recentlySent } = await supabase
    .from('host_group_outreach')
    .select('group_id')
    .eq('host_id', hostId)
    .gte('sent_at', thirtyDaysAgo);

  const excludeIds = (recentlySent || []).map((r: any) => r.group_id);

  let query = supabase
    .from('groups')
    .select(`
      id, name, description, min_budget, target_neighborhood,
      move_in_date, move_in_flexible,
      members:group_members(
        user:profiles(id, full_name, avatar_url, occupation)
      ),
      linked_listing:listings(id, title, bedrooms, city)
    `)
    .eq('discoverable', true)
    .eq('target_city', city);

  if (excludeIds.length > 0) {
    query = query.not('id', 'in', `(${excludeIds.join(',')})`);
  }

  const { data, error } = await query.order('created_at', { ascending: false });
  if (error) throw error;

  return (data || []).map((g: any) => ({
    ...g,
    memberCount: g.members?.length ?? 0,
    memberLimit: (g.linked_listing?.bedrooms ?? 2) + 1,
  }));
}

/**
 * Report a proactive outreach message.
 */
export async function reportProactiveOutreach(
  outreachId: string,
  reason: string
): Promise<void> {
  await supabase.rpc('report_host_outreach', {
    p_outreach_id: outreachId,
    p_reason: reason,
  });
}
```

## Screen: `HostRenterGroupsScreen.tsx`

```tsx
import React, { useEffect, useState, useCallback } from 'react';
import {

} from '../../services/hostOutreachService';
import { UNLOCK_PACKAGES } from '../../constants/planLimits';

export default function HostRenterGroupsScreen({ navigation }: any) {
  const { theme } = useTheme();
  const { user } = useAuth();
  const { initPaymentSheet, presentPaymentSheet } = useStripe();
  const plan = user?.subscription?.plan ?? 'starter';

  const [groups, setGroups] = useState<any[]>([]);
  const [loading, setLoading] = useState(true);
  const [city, setCity] = useState('New York');
  const [quota, setQuota] = useState<OutreachQuotaStatus | null>(null);

  // Compose modal state
  const [composeTarget, setComposeTarget] = useState<any | null>(null);
  const [message, setMessage] = useState('');
  const [sending, setSending] = useState(false);

  // Unlock modal state
  const [showUnlock, setShowUnlock] = useState(false);
  const [purchasing, setPurchasing] = useState<string | null>(null);

  useEffect(() => {
    loadData();
  }, [city]);

  async function loadData() {
    setLoading(true);
    try {
      const [groupData, quotaData] = await Promise.all([
        getDiscoverableGroups(city, user!.id),
        getOutreachQuotaStatus(user!.id, plan),
      ]);
      setGroups(groupData);
      setQuota(quotaData);
    } finally {
      setLoading(false);
    }
  }

  // ─── Send Message ─────────────────────────────────────────────
  async function handleSend() {
    if (!composeTarget) return;

    if (message.trim().length < 50) {
      Alert.alert('Too Short', 'Your message must be at least 50 characters to prevent spam.');
      return;
    }

    setSending(true);
    try {
      await sendProactiveOutreach(composeTarget.id, message.trim(), plan);
      setComposeTarget(null);
      setMessage('');
      // Refresh quota + remove messaged group from list
      const newQuota = await getOutreachQuotaStatus(user!.id, plan);
      setQuota(newQuota);
      setGroups(prev => prev.filter(g => g.id !== composeTarget.id));
      Alert.alert('✅ Message Sent', 'Your message was delivered to the group.');
    } catch (e: any) {
      const msg = e.message || '';
      if (msg === 'DAILY_LIMIT_REACHED') {
        setComposeTarget(null);
        setShowUnlock(true);
      } else if (msg === 'HOURLY_LIMIT_REACHED') {
        Alert.alert('Slow Down', 'You can only send a few messages per hour. Try again shortly.');
      } else if (msg === 'GROUP_COOLDOWN') {
        Alert.alert('Already Contacted', 'You messaged this group recently. Wait 30 days before reaching out again.');
      } else if (msg === 'MESSAGE_TOO_SHORT') {
        Alert.alert('Too Short', 'Write at least 50 characters.');
      } else if (msg === 'SUSPENDED') {
        Alert.alert('Outreach Suspended', 'Your outreach privileges have been suspended due to reports. Contact support to appeal.');
      } else {
        Alert.alert('Error', 'Could not send message. Please try again.');
      }
    } finally {
      setSending(false);
    }
  }

  // ─── Pay to Unlock ────────────────────────────────────────────
  async function handleUnlock(packageId: string) {
    setPurchasing(packageId);
    const pkg = UNLOCK_PACKAGES.find(p => p.id === packageId)!;

    try {
      const { data } = await fetch('/functions/v1/create-proactive-outreach-payment', {
        method: 'POST',
        headers: { 'Content-Type': 'application/json', Authorization: `Bearer ${user?.accessToken}` },
        body: JSON.stringify({ packageId }),
      }).then(r => r.json());

      const { error: initError } = await initPaymentSheet({
        paymentIntentClientSecret: data.clientSecret,
        merchantDisplayName: 'RoomDrx',
      });
      if (initError) throw new Error(initError.message);

      const { error: presentError } = await presentPaymentSheet();
      if (presentError) {
        if (presentError.code !== 'Canceled') Alert.alert('Payment Failed', presentError.message);
        return;
      }

      // Payment succeeded — credits added via webhook
      Alert.alert(
        '✅ Unlocked!',
        `${pkg.credits} additional messages added for today.`,
        [{ text: 'Send Messages', onPress: () => setShowUnlock(false) }]
      );

      // Refresh quota
      const newQuota = await getOutreachQuotaStatus(user!.id, plan);
      setQuota(newQuota);
    } catch (e: any) {
      Alert.alert('Error', e.message || 'Payment failed.');
    } finally {
      setPurchasing(null);
    }
  }

  const canSend = quota?.canSend ?? false;
  const isStarter = plan === 'starter';

  return (
    <View style={{ flex: 1, backgroundColor: theme.background }}>
      {/* ── Header ─────────────────────────────────────────────── */}
      <View style={styles.header}>
        <View>
          <ThemedText style={Typography.h2}>Renter Groups</ThemedText>
          <ThemedText style={[Typography.small, { color: theme.textSecondary }]}>
            {groups.length} groups looking for a place
          </ThemedText>
        </View>
        <View style={[styles.totalBadge, { backgroundColor: theme.primary + '20', borderColor: theme.primary + '40' }]}>
          <Feather name="users" size={13} color={theme.primary} />
          <ThemedText style={[Typography.small, { color: theme.primary, fontWeight: '700', marginLeft: 4 }]}>
            {groups.length}
          </ThemedText>
        </View>
      </View>

      {/* ── Quota Bar ──────────────────────────────────────────── */}
      {!isStarter && quota && (
        <View style={[styles.quotaBar, { backgroundColor: theme.card, borderColor: theme.border }]}>
          <View style={{ flex: 1 }}>
            <ThemedText style={[Typography.small, { fontWeight: '600' }]}>
              {quota.hitDailyLimit
                ? '🔒 Daily limit reached'
                : `${quota.remaining} message${quota.remaining !== 1 ? 's' : ''} left today`}
            </ThemedText>
            {quota.paidExtra > 0 && (
              <ThemedText style={[Typography.small, { color: '#22C55E' }]}>
                +{quota.paidExtra} paid credits active
              </ThemedText>
            )}
          </View>

          {quota.hitDailyLimit ? (
            <TouchableOpacity
              style={[styles.unlockBtn, { backgroundColor: theme.primary }]}
              onPress={() => setShowUnlock(true)}
            >
              <Feather name="zap" size={12} color="#fff" />
              <ThemedText style={[Typography.small, { color: '#fff', fontWeight: '700', marginLeft: 4 }]}>
                Unlock More
              </ThemedText>
            </TouchableOpacity>
          ) : (
            <View style={[styles.quotaTrack, { backgroundColor: theme.border }]}>
              <View
                style={[
                  styles.quotaFill,
                  {
                    backgroundColor: theme.primary,
                    width: `${Math.min(((quota.used) / (quota.dailyCap + quota.paidExtra)) * 100, 100)}%`,
                  }
                ]}

      )}

      {/* ── City Filter ────────────────────────────────────────── */}
      <View style={[styles.cityRow, { backgroundColor: theme.card, borderColor: theme.border }]}>
        <Feather name="map-pin" size={14} color={theme.primary} />
        <ThemedText style={[Typography.body, { flex: 1, marginLeft: 8, fontWeight: '600' }]}>
          {city}
        </ThemedText>
        <Feather name="chevron-down" size={16} color={theme.textSecondary} />
      </View>

      {/* ── Groups List ────────────────────────────────────────── */}
      {loading ? (
        <ActivityIndicator color={theme.primary} style={{ marginTop: 40 }} />
      ) : (
        <FlatList
          data={groups}
          keyExtractor={item => item.id}
          contentContainerStyle={{ padding: Spacing.md, paddingBottom: 100 }}
          renderItem={({ item }) => (
            <RenterGroupCard
              group={item}
              theme={theme}
              canMessage={canSend && !isStarter}
              onMessagePress={() => {
                if (quota?.hitDailyLimit) {
                  setShowUnlock(true);
                } else {
                  setComposeTarget(item);
                }
              }}
              onReportPress={(outreachId: string) => {
                Alert.alert('Report Message', 'Is this message spam or inappropriate?', [
                  { text: 'Cancel', style: 'cancel' },
                  { text: 'Report', style: 'destructive', onPress: () => reportProactiveOutreach(outreachId, 'Reported by group member') },
                ]);
              }}
            />
          )}
        />
      )}

      {/* ── Compose Modal ──────────────────────────────────────── */}
      <Modal
        visible={!!composeTarget}
        transparent
        animationType="slide"
        onRequestClose={() => setComposeTarget(null)}
      >
        <View style={styles.modalOverlay}>
          <View style={[styles.composeSheet, { backgroundColor: theme.card, borderColor: theme.border }]}>
            <View style={styles.composeHeader}>
              <ThemedText style={[Typography.h3]}>Message Group</ThemedText>
              <TouchableOpacity onPress={() => setComposeTarget(null)}>
                <Feather name="x" size={22} color={theme.text} />
              </TouchableOpacity>
            </View>

            <ThemedText style={[Typography.small, { color: theme.textSecondary, marginBottom: Spacing.sm }]}>
              To: <ThemedText style={{ fontWeight: '700', color: theme.text }}>{composeTarget?.name}</ThemedText>
            </ThemedText>

            <TextInput
              value={message}
              onChangeText={setMessage}
              multiline
              numberOfLines={5}
              placeholder="Tell this group about your listing. Be specific — mention the address, price, and move-in date. Minimum 50 characters."
              placeholderTextColor={theme.textSecondary}
              style={[styles.composeInput, { backgroundColor: theme.background, borderColor: theme.border, color: theme.text }]}
              maxLength={500}
            />

            {/* Character count + minimum warning */}
            <View style={styles.charRow}>
              <ThemedText style={[Typography.small, {
                color: message.length < 50 ? '#FF6B6B' : theme.textSecondary
              }]}>
                {message.length < 50
                  ? `${50 - message.length} more characters needed`
                  : `${message.length}/500`}
              </ThemedText>
            </View>

            {/* Quota warning */}
            {quota && quota.remaining <= 1 && !quota.hitDailyLimit && (
              <View style={[styles.warningRow, { backgroundColor: '#FF6B6B15', borderColor: '#FF6B6B30' }]}>
                <Feather name="alert-circle" size={13} color="#FF6B6B" />
                <ThemedText style={[Typography.small, { color: '#FF6B6B', marginLeft: 6 }]}>
                  Last message for today. Sending this uses your final credit.
                </ThemedText>
              </View>
            )}

            <TouchableOpacity
              style={[
                styles.sendBtn,
                { backgroundColor: message.length >= 50 && !sending ? theme.primary : theme.border }
              ]}
              onPress={handleSend}
              disabled={message.length < 50 || sending}
            >
              {sending
                ? <ActivityIndicator color="#fff" size="small" />
                : <>
                    <Feather name="send" size={16} color="#fff" />
                    <ThemedText style={[Typography.body, { color: '#fff', fontWeight: '700', marginLeft: 8 }]}>
                      Send Message
                    </ThemedText>
                  </>
              }
            </TouchableOpacity>
          </View>
        </View>
      </Modal>

      {/* ── Unlock Modal ───────────────────────────────────────── */}
      <Modal
        visible={showUnlock}
        transparent
        animationType="fade"
        onRequestClose={() => setShowUnlock(false)}
      >
        <View style={styles.modalOverlay}>
          <View style={[styles.unlockSheet, { backgroundColor: theme.card, borderColor: theme.border }]}>
            <View style={[styles.unlockIcon, { backgroundColor: theme.primary + '20' }]}>
              <Feather name="zap" size={24} color={theme.primary} />
            </View>

            <ThemedText style={[Typography.h3, { textAlign: 'center', marginTop: Spacing.sm }]}>
              Daily Limit Reached
            </ThemedText>
            <ThemedText style={[Typography.body, { color: theme.textSecondary, textAlign: 'center', marginTop: 4, marginBottom: Spacing.lg }]}>
              Purchase additional sends for today. Unused credits expire at midnight.
            </ThemedText>

            {UNLOCK_PACKAGES.map(pkg => (
              <TouchableOpacity
                key={pkg.id}
                style={[styles.unlockPackage, { backgroundColor: theme.background, borderColor: theme.border }]}
                onPress={() => handleUnlock(pkg.id)}
                disabled={!!purchasing}
              >
                <View style={{ flex: 1 }}>
                  <ThemedText style={[Typography.body, { fontWeight: '700' }]}>
                    {pkg.label}
                  </ThemedText>
                  <ThemedText style={[Typography.small, { color: theme.textSecondary }]}>
                    Valid today only · expires midnight
                  </ThemedText>
                </View>
                {purchasing === pkg.id
                  ? <ActivityIndicator color={theme.primary} size="small" />
                  : <ThemedText style={[Typography.h3, { color: theme.primary }]}>
                      ${(pkg.priceCents / 100).toFixed(2)}
                    </ThemedText>
                }
              </TouchableOpacity>
            ))}

            <TouchableOpacity
              style={styles.cancelUnlock}
              onPress={() => setShowUnlock(false)}
            >
              <ThemedText style={[Typography.body, { color: theme.textSecondary }]}>Maybe Later</ThemedText>
            </TouchableOpacity>
          </View>
        </View>
      </Modal>
    </View>
  );
}

// ─── Renter Group Card ───────────────────────────────────────────────────────

function RenterGroupCard({ group, theme, canMessage, onMessagePress, onReportPress }: any) {
  return (
    <View style={[styles.groupCard, { backgroundColor: theme.card, borderColor: theme.border }]}>
      {/* Member avatars */}
      <View style={styles.avatarRow}>
        {group.members.slice(0, 3).map((m: any, i: number) => (
          <View
            key={m.user?.id ?? i}
            style={[styles.avatar, { backgroundColor: theme.primary, borderColor: theme.card, marginLeft: i === 0 ? 0 : -10, zIndex: 3 - i }]}
          >
            <ThemedText style={{ fontSize: 14, fontWeight: '800', color: '#fff' }}>
              {m.user?.full_name?.charAt(0) ?? '?'}
            </ThemedText>
          </View>
        ))}
        {/* Empty slots */}
        {Array.from({ length: Math.max(0, (group.memberLimit ?? 3) - group.memberCount) }).map((_, i) => (
          <View
            key={`empty-${i}`}
            style={[styles.avatar, styles.emptyAvatar, { borderColor: theme.border, marginLeft: -10 }]}
          >
            <Feather name="plus" size={12} color={theme.textSecondary} />
          </View>
        ))}
      </View>

      {/* Fill status */}
      <ThemedText style={[Typography.small, { color: theme.textSecondary, marginTop: 8, textAlign: 'center' }]}>
        {group.memberCount} of {group.memberLimit} members filled{' '}
        <ThemedText style={[{ color: '#22C55E', fontWeight: '700' }]}>
          {group.memberLimit - group.memberCount} left
        </ThemedText>
      </ThemedText>

      {/* Name */}
      <ThemedText style={[Typography.h3, { textAlign: 'center', marginTop: 6 }]}>
        {group.name}
      </ThemedText>

      {/* Description */}
      <ThemedText style={[Typography.small, { color: theme.textSecondary, textAlign: 'center', marginTop: 4 }]} numberOfLines={2}>
        {group.description}
      </ThemedText>

      {/* Budget + Location chips */}
      <View style={styles.chipRow}>
        {group.min_budget && (
          <View style={[styles.chip, { backgroundColor: theme.primary + '15', borderColor: theme.primary + '30' }]}>
            <Feather name="dollar-sign" size={11} color={theme.primary} />
            <ThemedText style={[styles.chipText, { color: theme.primary }]}>
              ${group.min_budget.toLocaleString()}/mo
            </ThemedText>
          </View>
        )}
        {group.target_neighborhood && (
          <View style={[styles.chip, { backgroundColor: theme.border + '60', borderColor: theme.border }]}>
            <Feather name="map-pin" size={11} color={theme.textSecondary} />
            <ThemedText style={[styles.chipText, { color: theme.textSecondary }]}>
              {group.target_neighborhood}
            </ThemedText>
          </View>
        )}
      </View>

      {/* Move-in date */}
      {(group.move_in_date || group.move_in_flexible) && (
        <View style={styles.moveInRow}>
          <Feather name="calendar" size={12} color={theme.textSecondary} />
          <ThemedText style={[Typography.small, { color: theme.textSecondary, marginLeft: 6 }]}>
            Move-in: {group.move_in_flexible ? 'Flexible' : group.move_in_date}
          </ThemedText>
        </View>
      )}

      {/* Message button */}
      <TouchableOpacity
        style={[
          styles.messageBtn,
          { backgroundColor: canMessage ? theme.primary : theme.border }
        ]}
        onPress={onMessagePress}
        disabled={!canMessage}
      >
        <Feather name="send" size={15} color="#fff" />
        <ThemedText style={[Typography.body, { color: '#fff', fontWeight: '700', marginLeft: 8 }]}>
          {canMessage ? 'Message This Group' : 'Upgrade to Message'}
        </ThemedText>
      </TouchableOpacity>
    </View>
  );
}
```

## Summary of All Spam Controls

| Control | Rule | What Happens When Hit |
|---------|------|-----------------------|
| **Daily cap** | Pro: 3/day · Business: 10/day | Unlock modal appears — pay for more |
| **Paid unlock expires** | Credits expire at midnight | Cannot stockpile or save for tomorrow |
| **Hourly rate limit** | Pro: 2/hr · Business: 3/hr | Error toast — try again later |
| **Per-group cooldown** | 30 days per group | Group hidden from list for 30 days |
| **Minimum length** | 50 characters | Send button disabled + counter shown |
| **Report threshold** | 3 reports = auto-suspended | Outreach locked — appeal via support |
| **No rollover** | Monthly credits don't roll over | Fresh slate each month |
