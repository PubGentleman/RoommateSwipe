**REPLIT PROMPT — Reveal Full Listing Address Only After Host Accepts Inquiry**

The full property address is currently stored in the database but never shared with renters at the right moment. Add a flow where the full address is only revealed to a renter once the host accepts their inquiry. Before acceptance, renters always see neighborhood + city only.

Add an inquiry status field to track whether a host has accepted a renter's inquiry:

```sql
  address_revealed boolean default false;

alter table public.matches add column if not exists 
  inquiry_status text check (
    inquiry_status in ('pending', 'accepted', 'declined')
  ) default 'pending';
```

**PART 2 — UPDATE LOCATION DISPLAY LOGIC**

In `utils/matchingAlgorithm.ts`, update the `formatLocation()` function to accept a `revealed` parameter:

```ts
export const formatLocation = (
  listing: Listing,
  revealed: boolean = false
): string => {
  if (revealed && listing.address) {
    return `${listing.address}, ${listing.city}, ${listing.state}`;
  }
  if (listing.neighborhood) {
    return `${listing.neighborhood}, ${listing.city}`;
  }
  return `${listing.city}, ${listing.state}`;
};
```

Pass `revealed: false` everywhere the listing is shown publicly. Only pass `revealed: true` after the host accepts the inquiry.

**PART 7 — SERVICES UPDATE**

Update `services/matchService.ts`:

```ts
export const acceptInquiry = async (matchId: string) =>
  supabase
    .from('matches')
    .update({ 
      inquiry_status: 'accepted', 
      address_revealed: true,
      updated_at: new Date().toISOString()
    })
    .eq('id', matchId);

export const declineInquiry = async (matchId: string) =>
  supabase
    .from('matches')
    .update({ 
      inquiry_status: 'declined',
      updated_at: new Date().toISOString()
    })
    .eq('id', matchId);

export const isAddressRevealed = async (matchId: string): Promise<boolean> => {
  const { data } = await supabase
    .from('matches')
    .select('address_revealed')
    .eq('id', matchId)
    .single();
  return data?.address_revealed ?? false;
};
```
