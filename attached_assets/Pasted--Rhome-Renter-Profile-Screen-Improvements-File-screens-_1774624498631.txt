## Prompt 1 — Make Lifestyle Tags Self-Explanatory

### Bug
The Lifestyle section shows terse labels like "Office", "Very Social", "Very Clean" with no context. A new user has no idea what "Office" means in terms of living together. Each tag needs a human-readable subtitle that explains its roommate relevance.

### Fix

**Step 1 — Replace the lifestyle pill array with a richer structure that includes a subtitle per tag.**

Find the Lifestyle section in the profile detail modal (inside `showProfileDetail`, look for `<Text style={styles.pdSectionLabel}>Lifestyle</Text>`). Replace the entire `<View style={styles.pdPillWrap}>` block with:

```tsx
<View style={{ gap: 8 }}>
  {([
    {
      icon: 'star' as const,
      value: getCleanlinessLabel(currentProfile.lifestyle?.cleanliness),
      sub: (() => {
        const v = currentProfile.lifestyle?.cleanliness ?? 3;
        if (v >= 4.5) return 'Keeps shared spaces spotless';
        if (v >= 3.5) return 'Tidies up regularly';
        if (v >= 2.5) return 'Moderately tidy';
        if (v >= 1.5) return 'Relaxed about mess';
        return 'Very laid-back about cleanliness';
      })(),
      color: '#ff6b5b',
    },
    {
      icon: 'users' as const,
      value: getSocialLevelLabel(currentProfile.lifestyle?.socialLevel),
      sub: (() => {
        const v = currentProfile.lifestyle?.socialLevel ?? 3;
        if (v >= 4.5) return 'Loves having people over';
        if (v >= 3.5) return 'Enjoys socializing at home';
        if (v >= 2.5) return 'Balanced — social but values quiet time';
        if (v >= 1.5) return 'Prefers a quieter home';
        return 'Needs a very quiet environment';
      })(),
      color: '#ff6b5b',
    },
    {
      icon: 'briefcase' as const,
      value: getWorkScheduleLabel(currentProfile.lifestyle?.workSchedule),
      sub: (() => {
        const s = currentProfile.lifestyle?.workSchedule;
        if (s === 'wfh' || s === 'wfh_fulltime' || s === 'remote') return 'Home most of the day';
        if (s === 'office' || s === 'office_fulltime') return 'Out of the apartment 9–5';
        if (s === 'hybrid') return 'Mix of home and office days';
        if (s === 'night_shift') return 'Works nights — sleeps during the day';
        if (s === 'irregular') return 'Varied schedule — hours change often';
        if (s === 'student') return 'On campus most days';
        if (s === 'freelance') return 'Works from home on flexible hours';
        return 'Schedule varies';
      })(),
      color: '#ff6b5b',
    },
    currentProfile.lifestyle?.pets
      ? { icon: 'heart' as const, value: 'Has Pets', sub: 'Will bring a pet to the apartment', color: '#f39c12' }
      : null,
    currentProfile.lifestyle?.smoking
      ? { icon: 'wind' as const, value: 'Smoking OK', sub: 'Open to smoking in shared spaces', color: 'rgba(255,255,255,0.5)' }
      : { icon: 'x' as const, value: 'No Smoking', sub: 'Prefers a smoke-free home', color: 'rgba(255,255,255,0.35)' },
    currentProfile.preferences?.privateBathroom
      ? { icon: 'droplet' as const, value: 'Needs Private Bath', sub: 'Requires their own bathroom', color: '#ff6b5b' }
      : null,
  ] as (null | { icon: any; value: string; sub: string; color: string })[])
    .filter(Boolean)
    .map((tag, i) => (
      <View key={i} style={{
        flexDirection: 'row',
        alignItems: 'flex-start',
        gap: 12,
        paddingVertical: 10,
        paddingHorizontal: 14,
        backgroundColor: 'rgba(255,255,255,0.04)',
        borderRadius: 12,
        borderWidth: 1,
        borderColor: 'rgba(255,255,255,0.07)',
      }}>
        <View style={{
          width: 32, height: 32, borderRadius: 10,
          backgroundColor: 'rgba(255,107,91,0.1)',
          alignItems: 'center', justifyContent: 'center',
          marginTop: 1,
        }}>
          <Feather name={tag!.icon} size={14} color={tag!.color} />
        </View>
        <View style={{ flex: 1 }}>
          <Text style={{ fontSize: 14, fontWeight: '600', color: '#FFFFFF' }}>
            {tag!.value}
          </Text>
          <Text style={{ fontSize: 12, color: 'rgba(255,255,255,0.45)', marginTop: 2 }}>
            {tag!.sub}
          </Text>
        </View>
      </View>
    ))}
</View>
```

## Prompt 2 — Add "Why This Match" + "Ask AI" to the Profile Detail Modal

### Bug
The "Why?" and "Ask AI" buttons exist only on the swipe card overlay. When a user opens the full expanded profile (`showProfileDetail = true`), there are no AI buttons anywhere — only Pass, I'm Interested, and Message at the bottom. The profile detail is exactly the moment users need AI context most, and it's not available there.

### Fix

**Step 1 — Add an AI action row above the bottom action bar in the profile detail modal.**

Find the `<View style={[styles.pdActionBar, ...]}>`  at the bottom of the profile detail modal (around line 2607). Just BEFORE that entire `pdActionBar` view, insert this AI row:

```tsx
{/* AI Actions Row */}
<View style={{
  flexDirection: 'row',
  gap: 10,
  paddingHorizontal: 16,
  paddingTop: 10,
  paddingBottom: 6,
  backgroundColor: '#111',
}}>
  {/* Why This Match */}
  {renterLimits.hasMatchBreakdown ? (
    <Pressable
      style={{
        flex: 1,
        flexDirection: 'row',
        alignItems: 'center',
        justifyContent: 'center',
        gap: 6,
        paddingVertical: 11,
        borderRadius: 12,
        backgroundColor: 'rgba(255,107,91,0.1)',
        borderWidth: 1,
        borderColor: 'rgba(255,107,91,0.2)',
      }}
      onPress={() => setShowWhyModal(true)}
    >
      <Feather name="zap" size={14} color="#ff6b5b" />
      <Text style={{ fontSize: 13, fontWeight: '600', color: '#ff6b5b' }}>
        Why this match?
      </Text>
    </Pressable>
  ) : (
    <Pressable
      style={{
        flex: 1,
        flexDirection: 'row',
        alignItems: 'center',
        justifyContent: 'center',
        gap: 6,
        paddingVertical: 11,
        borderRadius: 12,
        backgroundColor: 'rgba(168,85,247,0.1)',
        borderWidth: 1,
        borderColor: 'rgba(168,85,247,0.2)',
      }}
      onPress={() => {
        setShowProfileDetail(false);
        setTimeout(() => (navigation as any).navigate('Plans'), 200);
      }}
    >
      <Feather name="lock" size={13} color="#a855f7" />
      <Text style={{ fontSize: 13, fontWeight: '600', color: '#a855f7' }}>
        Why this match?
      </Text>
    </Pressable>
  )}

  {/* Ask AI */}
  <Pressable
    style={{
      flex: 1,
      flexDirection: 'row',
      alignItems: 'center',
      justifyContent: 'center',
      gap: 6,
      paddingVertical: 11,
      borderRadius: 12,
      backgroundColor: 'rgba(255,107,91,0.08)',
      borderWidth: 1,
      borderColor: 'rgba(255,107,91,0.18)',
    }}
    onPress={() => {
      setShowProfileDetail(false);
      setTimeout(() => {
        setAskAboutTarget({
          id: currentProfile.id,
          name: currentProfile.name,
          age: currentProfile.age,
          compatibility: currentProfile.compatibility,
          entryPoint: 'match_screen',
        });
        setAskAboutVisible(true);
      }, 300);
    }}
  >
    <Feather name="cpu" size={14} color="#ff6b5b" />
    <Text style={{ fontSize: 13, fontWeight: '600', color: '#ff6b5b' }}>
      Ask AI about {currentProfile.name?.split(' ')[0] || 'them'}
    </Text>
  </Pressable>
</View>
```

No new state variables needed — `showWhyModal`, `askAboutTarget`, `askAboutVisible` are all already declared in the component.

## Prompt 3 — Photo Grid: "See All Photos" Tap-to-Expand

### Bug
Photos in the profile detail are displayed as a horizontal scroll (one photo at a time with dot indicators). There's no way to scan all photos at once. Users who want a quick overview of someone's photos have to swipe through them one by one.

### Fix

**Step 1 — Add state for the photo grid modal.**

Add near the other modal state variables at the top of the component (around line 89):
```ts
const [showPhotoGrid, setShowPhotoGrid] = useState(false);
const [photoGridIndex, setPhotoGridIndex] = useState(0);
```

**Step 2 — Add a "See all X photos" button overlay on the photo area in the profile detail modal.**

Find the photo dots section in the profile detail (the block starting with `{photosArray.length > 1 ? (`). After the dots view and before the close button (`pdCloseBtn`), add:

```tsx
{photosArray.length > 1 ? (
  <Pressable
    style={{
      position: 'absolute',
      bottom: 16,
      right: 16,
      flexDirection: 'row',
      alignItems: 'center',
      gap: 5,
      backgroundColor: 'rgba(0,0,0,0.55)',
      paddingHorizontal: 10,
      paddingVertical: 6,
      borderRadius: 20,
      borderWidth: 1,
      borderColor: 'rgba(255,255,255,0.15)',
    }}
    onPress={() => {
      setPhotoGridIndex(currentPhotoIndex);
      setShowPhotoGrid(true);
    }}
  >
    <Feather name="grid" size={12} color="#FFFFFF" />
    <Text style={{ color: '#FFFFFF', fontSize: 12, fontWeight: '600' }}>
      {photosArray.length} photos
    </Text>
  </Pressable>
) : null}
```

**Step 3 — Add the full-screen photo grid modal.**

Add this modal just after the profile detail modal's closing `</Modal>` tag (around line 2665):

```tsx
{/* Photo Grid Modal */}
<Modal
  visible={showPhotoGrid}
  animationType="fade"
  transparent={false}
  onRequestClose={() => setShowPhotoGrid(false)}
>
  <View style={{ flex: 1, backgroundColor: '#000' }}>
    {/* Header */}
    <View style={{
      flexDirection: 'row',
      alignItems: 'center',
      justifyContent: 'space-between',
      paddingHorizontal: 16,
      paddingTop: Math.max(16, insets.top + 8),
      paddingBottom: 12,
    }}>
      <Text style={{ color: '#FFFFFF', fontSize: 16, fontWeight: '700' }}>
        {currentProfile?.name?.split(' ')[0]}'s Photos
      </Text>
      <Pressable
        onPress={() => setShowPhotoGrid(false)}
        style={{
          width: 32, height: 32, borderRadius: 16,
          backgroundColor: 'rgba(255,255,255,0.1)',
          alignItems: 'center', justifyContent: 'center',
        }}
      >
        <Feather name="x" size={18} color="#fff" />
      </Pressable>
    </View>

    {(() => {
      const photosArray = Array.isArray(currentProfile?.photos)
        ? currentProfile.photos
        : currentProfile?.photos
        ? [currentProfile.photos]
        : [];

      return (
        <>
          {/* Selected photo fullscreen view */}
          <View style={{ width: '100%', height: 340, marginBottom: 12 }}>
            <Image
              source={{ uri: photosArray[photoGridIndex] }}
              style={{ width: '100%', height: '100%' }}
              resizeMode="cover"
            />
            <View style={{
              position: 'absolute',
              bottom: 10,
              right: 12,
              backgroundColor: 'rgba(0,0,0,0.55)',
              paddingHorizontal: 10,
              paddingVertical: 4,
              borderRadius: 12,
            }}>
              <Text style={{ color: '#fff', fontSize: 12, fontWeight: '600' }}>
                {photoGridIndex + 1} / {photosArray.length}
              </Text>
            </View>
          </View>

          {/* Thumbnail grid */}
          <ScrollView contentContainerStyle={{
            flexDirection: 'row',
            flexWrap: 'wrap',
            gap: 3,
            paddingHorizontal: 3,
          }}>
            {photosArray.map((uri, i) => (
              <Pressable
                key={i}
                onPress={() => setPhotoGridIndex(i)}
                style={{
                  width: '32.5%',
                  aspectRatio: 1,
                  borderRadius: 4,
                  overflow: 'hidden',
                  borderWidth: photoGridIndex === i ? 2 : 0,
                  borderColor: '#ff6b5b',
                }}
              >
                <Image
                  source={{ uri }}
                  style={{ width: '100%', height: '100%' }}
                  resizeMode="cover"
                />
              </Pressable>
            ))}
          </ScrollView>
        </>
      );
    })()}
  </View>
</Modal>
```
