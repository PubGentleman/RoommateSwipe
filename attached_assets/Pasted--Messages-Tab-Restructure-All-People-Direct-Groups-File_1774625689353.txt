# Messages Tab Restructure: All / People / Direct / Groups

**File:** `screens/shared/MessagesScreen.tsx`

**Goal:** Reorganize the 4 chat filter tabs so each tab has a clear, singular purpose:

| Tab | Shows |
|---|---|
| **All** | Everything (unchanged) |
| **People** | Mutual matches + cold/unmatched outreach (all 1-on-1 person chats) |
| **Direct** | Renter ↔ host conversations about a specific listing (`isInquiryThread: true`) |
| **Groups** | Multi-person roommate group entities from the Groups feature |

Cold/unmatched messages (`matchType: 'cold'`) move OUT of Direct and INTO People — with a visual "Direct" pill badge so users can still tell them apart from matched conversations.

## Step 6 — Update `filteredConversations` and inject a section divider for cold messages in People tab

Find the `filteredConversations` block:
```ts
const filteredConversations = (() => {
  let convs = conversations;
  if (searchQuery.trim()) {
    convs = convs.filter(c =>
      c.participant.name.toLowerCase().includes(searchQuery.toLowerCase()) ||
      (c.lastMessage && c.lastMessage.toLowerCase().includes(searchQuery.toLowerCase()))
    );
  }
  if (chatFilter !== 'all') {
    convs = convs.filter(c => getConvType(c) === chatFilter);
  }
  return convs;
})();
```

Replace with:
```ts
const DIVIDER_ITEM = { id: '__cold_divider__', __isDivider: true } as any;

const filteredConversations = (() => {
  let convs = conversations;
  if (searchQuery.trim()) {
    convs = convs.filter(c =>
      c.participant.name.toLowerCase().includes(searchQuery.toLowerCase()) ||
      (c.lastMessage && c.lastMessage.toLowerCase().includes(searchQuery.toLowerCase()))
    );
  }

  // For People tab (or All tab showing people), separate matched vs cold with a divider
  if (chatFilter === 'people' || chatFilter === 'all') {
    const peopleConvs = chatFilter === 'people'
      ? convs.filter(c => getConvType(c) === 'people')
      : convs;

    if (chatFilter === 'people') {
      const matched = peopleConvs.filter(c => c.matchType !== 'cold');
      const cold = peopleConvs.filter(c => c.matchType === 'cold');
      // Only inject divider if there are both matched AND cold conversations
      if (matched.length > 0 && cold.length > 0) {
        return [...matched, DIVIDER_ITEM, ...cold];
      }
      return peopleConvs;
    }
  }

  if (chatFilter !== 'all') {
    convs = convs.filter(c => getConvType(c) === chatFilter);
  }
  return convs;
})();
```
