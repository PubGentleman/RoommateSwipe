# RoomDrx — AI Match Assistant Screen Redesign

## File: `screens/renter/AIMatchAssistantScreen.tsx`
(may also be named `AIAssistantScreen.tsx` or `AIChatScreen.tsx`)

---

## Issues Being Fixed

1. **No chat bubble styling** — AI greeting is plain text in a box, not a chat message
2. **No AI avatar** — nothing distinguishes the AI's message visually from plain body text
3. **No chat input bar** — the screen has no visible text input to actually talk to the AI
4. **Massive dead space** — 4 chips then empty black screen for 60% of the height
5. **PREMIUM FEATURE badge** — awkward purple pill hanging under the title disconnected from everything
6. **Quick-action chips too plain** — small outlined chips with no icons, no visual weight
7. **Screen doesn't feel premium** — supposed to be a flagship AI feature, looks like a placeholder

## Change 5 — State Management for Chat

Add the necessary state variables if not already present:

```tsx
const [messages, setMessages] = useState<ChatMessage[]>([]);
const [inputText, setInputText] = useState('');
const [isTyping, setIsTyping] = useState(false);
const scrollRef = useRef<ScrollView>(null);

// Auto-scroll to bottom on new messages:
useEffect(() => {
  if (messages.length > 0) {
    setTimeout(() => scrollRef.current?.scrollToEnd({ animated: true }), 100);
  }
}, [messages, isTyping]);

const handleSend = async () => {
  if (!inputText.trim() || isTyping) return;
  const userMsg = { id: Date.now().toString(), role: 'user', text: inputText.trim() };
  setMessages(prev => [...prev, userMsg]);
  setInputText('');
  setIsTyping(true);
  // Call your AI service here (Supabase Edge Function or OpenAI):
  try {
    const response = await callAIAssistant(userMsg.text);
    setMessages(prev => [...prev, { id: Date.now().toString(), role: 'ai', text: response }]);
  } finally {
    setIsTyping(false);
  }
};

const handleQuickAction = (action: string) => {
  const prompts: Record<string, string> = {
    budget:  'Find me roommates within my budget',
    zodiac:  'Check my zodiac compatibility with potential roommates',
    hoods:   'What are the best neighborhoods for me in New York?',
    food:    'Recommend some restaurants near my area',
  };
  setInputText(prompts[action] ?? '');
};
```
