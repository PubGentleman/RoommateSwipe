# RoomDrx — Renter Profile Detail Modal Redesign

## File: `screens/renter/RoommatesScreen.tsx` or `screens/host/BrowseRenterGroupsScreen.tsx`
Look for the modal triggered by tapping a renter card — likely a `<Modal>` or bottom sheet with `showProfileDetail` state. The internal component may be named `ProfileDetailModal`, `RenterProfileModal`, or similar.

---

## Issues Being Fixed

1. **"Profile Details" header wastes space** — plain centered text + X; photo should be full-bleed with info overlaid
2. **Left/right arrow buttons on photo** — dated, should be swipeable with dot indicators
3. **Match score buried at bottom** — 85% match should be a badge near the name, not a bar at the very bottom
4. **Preference rows are clinical** — plain icon/label/value list reads like a spreadsheet
5. **Lifestyle values are plain text** — "Very Clean", "Very Social" should be visual pills
6. **"Fully Verified" badge is teal/blue** — should use the coral/green accent system
7. **No action buttons** — no way to message or pass from the modal; only the match score bar exists
8. **Zodiac shown as a colored square** — needs to be a styled text pill with the sign name

---

## Change 1 — Full-Bleed Photo Header with Overlaid Info

Remove the plain "Profile Details" title bar. Let the photo take the full top of the modal with name, age, match score, and verified status overlaid via a gradient.

```tsx
// Add state for photo index:
const [photoIdx, setPhotoIdx] = useState(0);

// Reset when opening new profile:
useEffect(() => { setPhotoIdx(0); }, [selectedProfile?.id]);

// Replace the header + photo section with:
<View style={styles.pdPhotoWrap}>
  {/* Swipeable photo */}
  <ScrollView
    horizontal
    pagingEnabled
    showsHorizontalScrollIndicator={false}
    onMomentumScrollEnd={(e) => {
      const idx = Math.round(e.nativeEvent.contentOffset.x / SCREEN_WIDTH);
      setPhotoIdx(idx);
    }}
  >
    {(profile.photos ?? [profile.photo]).map((uri, i) => (
      <Image
        key={i}
        source={{ uri }}
        style={styles.pdPhoto}
        resizeMode="cover"
      />
    ))}
  </ScrollView>

  {/* Gradient overlay — bottom of photo */}
  <LinearGradient
    colors={['transparent', 'rgba(0,0,0,0.85)']}
    style={styles.pdPhotoGradient}
  >
    {/* Match score badge */}
    <View style={[
      styles.pdMatchBadge,
      { backgroundColor: profile.matchScore >= 70 ? '#2ecc71' : profile.matchScore >= 40 ? '#f39c12' : '#e74c3c' }
    ]}>
      <Feather name="heart" size={11} color="#fff" />
      <Text style={styles.pdMatchText}>{profile.matchScore}% Match</Text>
    </View>

    {/* Name + age */}
    <Text style={styles.pdName}>{profile.name}, {profile.age}</Text>

    {/* Occupation + verified */}
    <View style={styles.pdSubRow}>
      <Text style={styles.pdOccupation}>{profile.occupation}</Text>
      {profile.isVerified && (
        <View style={styles.pdVerifiedBadge}>
          <Feather name="check-circle" size={11} color="#2ecc71" />
          <Text style={styles.pdVerifiedText}>Verified</Text>
        </View>
      )}
      {profile.zodiacSign && (
        <View style={styles.pdZodiacPill}>
          <Text style={styles.pdZodiacText}>{profile.zodiacSign}</Text>
        </View>
      )}
    </View>
  </LinearGradient>

  {/* Dot indicators */}
  {(profile.photos ?? []).length > 1 && (
    <View style={styles.pdDots}>
      {(profile.photos ?? [profile.photo]).map((_, i) => (
        <View key={i} style={[styles.pdDot, i === photoIdx && styles.pdDotActive]} />
      ))}
    </View>
  )}

  {/* Close button */}
  <Pressable style={styles.pdCloseBtn} onPress={onClose}>
    <Feather name="x" size={18} color="#fff" />
  </Pressable>
</View>
```
